SYR_GBTDC_21


  • 1.1.29.10 SYR_GBTDC_21: Detect CC1 Voltage for GBT DC coupler (ILD_DC)
  • 1.1.29.10.1 Description
  • 1.1.29.10.1.0-1 Brief description: Detect Voltage at CC1 input Pin 28 (A_TCHA_PERM) at CCU external LV signal connector. CC1 is a shared pin with CHademo vehicle charge permission pin.
  • 1.1.29.10.1.0-2 Preconditions: CCU is ready to sense changes on CC1 pin.
  • 1.1.29.10.1.0-3 Trigger: Plug is inserted in 'DC ladedose' intelligent socket
  • 1.1.29.10.1.0-4 Input data: HW signal PIN28
  • 1.1.29.10.1.0-5 Description of behaviour: When the Plug is inserted in intelligent socket. The CCU will detect change in Pin 28 : GBT CC1. Appropiate Signal received from intelligent socket should be send directly to LIM-C via RTE interface BMW_u_Cc1DcChnChgrMeas.
  • 1.1.29.10.1.0-6 Timing Requirements: 10 ms cyclic.
  • 1.1.29.10.1.0-7 Output data: RTE signal BMW_u_Cc1DcChnChgrMeas.
