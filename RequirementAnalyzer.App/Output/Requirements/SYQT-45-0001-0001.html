#SYQT-45-0001-0001#

[Factory_Inspection],[Factory_Inspection],[Diag_jobs]
Test Objective: Enable assembly mode and check status

Requirements

  • 6848 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    In the assembly mode of the DCDC converter must be disabled.
  • 6956 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    It must be possible to set the assembly mode using a diagnostic request.
  • 6957 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    It must be possible to reset the assembly mode using a diagnostic request.
  • 6958 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    It must be possible to read out the status of the assembly mode per diagnostic request.
  • 6992 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT_EE (ENG.8)][Requirement]
    The component shall implement a test mode that can be controlled via diagnostic request, that allows finding a failure cause if the cause cannot be distinguised in normal operation (peripheral vs. internal defect).   • 7532 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The assembly mode shall support the states active and inactive.
  • 7543 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The assembly mode shall be active and completely operable in the delivery state - upon delivery of the control unit in the vehicle plant .
  • 7544 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    When the mounting mode is activated, the error memory entry "Mounting mode active" must be set (this is specified in the framework of DTC quantity creation by BMW).
  • 7545 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Upon executed deactivation of the assembly mode the error memory entry shall autonomously be deleted.
