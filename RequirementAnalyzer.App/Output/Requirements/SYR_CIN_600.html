SYR_CIN_600


  • 8.1.57 SYR_CIN_600 : Connect PDU INFO_DT_TARO_SLT_2 to coding Parameter SRAHardware
  • 8.1.57.1 Description
  • 8.1.57.1.0-1 Brief description: Coding parameter ‘SRAHardware’ will indicate the presence of MRR hardware ( MRR HW mounted or not mounted) When the hardware is mounted the RX PDU group INFO_DT_TARO_SLT_2 shall be activated or else it shall be deactivated. Corresponding timeout DTC ‘0xD794D8 ‘ shall not be logged if the PDU is deactivated .
  • 8.1.57.1.0-2 Preconditions: SRAHardware coding parameter value is available
  • 8.1.57.1.0-3 Trigger: NA
  • 8.1.57.1.0-4 Input data: CAF file , FA file
  • 8.1.57.1.0-5 Description of behaviour: when coding parameter 'SRAHarware' >0 DU INFO_DT_TARO_SLT_2 IN PDU group is activated & if PDU INFO_DT_TARO_SLT_2 is not received in time , DTC 0xD794D8 shall be set when coding parameter 'SRAHarware' = 0 IN PDU group is deactivated & if PDU INFO_DT_TARO_SLT_2 is not received , DTC 0xD794D8 shall NOT be set
  • 8.1.57.1.0-6
  • 8.1.57.1.0-7 Timing Requirements: NA
  • 8.1.57.1.0-8 Output data: DTC 0x D794D8 status depending on the coding parameter SRAHardware value & Reception of INFO_DT_TRO_SLT_2
