  • 3.1.42 SYR_HMG105: DTC 0x224111 "Cooling-pump Power Electronics -Detection of Coolant loss"
  • 3.1.42.1 Description
  • 3.1.42.1.0-1 Brief description: the CCU has to be able to detect the failure "Detection of Coolant loss" of the Waterpump EWP23 (Cooling-pump of the Power Electronics)
  • 3.1.42.1.0-2 Preconditions: Kl15 ON and LIN Communication between the Waterpump EWP23 and the LIN-Master CCU is correctly working
  • 3.1.42.1.0-3 Trigger: ST_EWP23_LIN.ERR_ST_DRYR_EWP23_LIN=0x01;
  • 3.1.42.1.0-4 Input data: LIN Signal ERR_ST_DRYR_EWP23_LIN
  • 3.1.42.1.0-5 Description of behaviour: The Waterpump EWP23 will signalize to the CCU via LIN the presence of the failure "Detection of Coolant loss". The failure will be detected when the signal ERR_ST_DRYR_EWP23_LIN=0x01. As soon as this error condition is detected (after the expiration of a debouncing time whose value has still to be provided by BMW) an entry with Code 0x224111 will be written in the Error Memory
  • 3.1.42.1.0-6 Timing Requirements: Debouncing Time----> 5000 ms (starting from the 13th message) Healing Time ----> 2000 ms
  • 3.1.42.1.0-7 Output data: the DTC with code 0x224111

  • 3.1.42.1.0-10 Dependencies and interactions: dependency to: a. the LIN signal ERR_ST_DRYR_EWP23_LIN b. LIN Communication (in case of physical failures on LIN this failure cannot be detected)
