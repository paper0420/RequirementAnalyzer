  • 2.1.15.1.0-10 * Detect current over the maximum current threshold on the AC side: HVAC_Curr > Threshold_MaximumCurrent_AC * The Charger should stop operation -> msg PMG/LDK * Register the respective DTC and set bit "Abschaltung Maximal-strom AC" in the error status word * Implement as ASIL B. With Mean error probability: 10 FIT outage metrics: SPFM: > 90% ; LFM: > 60% implementation hint if The AC charging current (rms) (about 100 ms) per AC phase (AC / DC converter) > Maximum allowed Limits(Configurable) then The AC charging must be aborted. configurable limits: For the 3.5kW variant it is provided 16 A per Phase For the 7.0kW variant it is provided 32 A per Phase For the 11kW variant it is provided 16 A per Phase DTC Codes: Bauteilschutz - AC Überstrom Modul 1 0x318E2 Modul 2 0x318E9 Modul 3 0x318F1
