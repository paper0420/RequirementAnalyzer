SYR_ACC100


  • 2.1.11 SYR_ACC100: Component protection -Detect and handle undervoltage on AC HV
  • 2.1.11.1 Description
  • 2.1.11.1.0-1 Brief description:
  • 2.1.11.1.0-2 Detect and handle under voltage on the AC side, set error status bit "Abschaltung Unterspannung"
  • 2.1.11.1.0-3 Preconditions:
  • 2.1.11.1.0-4 LDK commands the target operation state "charge".
  • 2.1.11.1.0-5 Trigger:
  • 2.1.11.1.0-6 HVAC_Volt < Threshold_Undervoltage_AC. hint: e.g. Threshol_Undervoltage_AC = 70 V (KLH 18401)
  • 2.1.11.1.0-7 Input data:
  • 2.1.11.1.0-8 HVAC_U_L1Sig HVAC_U_L2Sig HVAC_U_L3Sig
  • 2.1.11.1.0-9 Description of behaviour:
  • 2.1.11.1.0-10 * Detect under voltage on the AC side: HVAC_Volt < Threshold_Undervoltage_AC * Check if error persists for more than Threshold_Undervoltage_AC_time * Register the respective DTC and set bit "Abschaltung Maximal-strom AC" in the error status word Implementation hint DTC Codes: (Bauteilschutz - AC Unterspannung) Modul 1 0x318E3 Modul 2 0x318EA Modul 3 0x318F2
  • 2.1.11.1.0-11 Timing Requirements:
  • 2.1.11.1.0-12 HVAC_Volt < Threshold_Undervoltage_AC and t_error > Threshold_Undervoltage_AC_time the error persists during a configurable time period (eg., 5 seconds).
  • 2.1.11.1.0-13 Output Data:
  • 2.1.11.1.0-14 "Abschaltung Unterspannung AC" Statuswort
  • 2.1.11.1.0-15 Postconditions:
  • 2.1.11.1.0-16 Descriptions of exceptions:
  • 2.1.11.1.0-17 If an error was found "Abschaltung Unterspannung AC-HV" bit shall be set in the error status word.
  • 2.1.11.1.0-18 Dependencies and interactions:
  • 2.1.11.1.0-19 Other modules/components LDK - Diagnostic customer requirements [18401 ,18403,18392] --> Error on undervoltage [23344] --> Interrupt on lossing intput voltage Usecases: SYR_ACC90
