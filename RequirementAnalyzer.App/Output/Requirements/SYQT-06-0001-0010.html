Requirements

  • 11449 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The development object must be able to charge the high voltage battery at the defined environmental conditions in chapter "Environmental Requirements" and at the defined range of input voltage and input frequency in chapter "Input Parameters".
  • 11450 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The development object must pass the test in chapter "Chinese Charging Function Test".
  • 11452 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) activate the development object at input voltage (± 15% of the rated input voltage) and input frequency (50 Hz ± 2%)
  • 11453 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) operate the development object at rated output power.
  • 17638 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) the development object must be able to charge the high voltage battery without causing any hazard against the high voltage battery or person.
  • 17984 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Definition: “Operate in a normal manner”/ "operate normally" The charging, communication, display and various protection functions of the development object must work without failures by validating the key parameters specified in "10.4.1 Parameter test (small)" in [95024-3-1].
  • 9860 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    With the input voltage frequency at 50 Hz ± 2% in accordance to table "Rated input voltage and rated input current", the development object must be able to operate in a normal manner.
  • 9866 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) adjust the input voltage of the development object to the rated voltage. b) activate the development object. c) operate the development object at rated load condition.
  • 9867 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) adjust input frequency of the development object to 49 Hz. b) remain at adjusted input frequency for 1 min. c) adjust input frequency of the development object to 51 Hz. d) remain at adjusted input frequency for 1 min.
  • 9868 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) the development object must work as described in chpter 3.5 during test by validating the key parameters specified in "10.4.1 Parameter test (small)" in [95024-3-1].
