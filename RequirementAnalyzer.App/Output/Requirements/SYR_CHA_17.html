SYR_CHA_17


  • 1.1.28.11 SYR_CHA_17: detect and process wakeup by CHAdeMO lines
  • 1.1.28.11.1 Description
  • 1.1.28.11.1.0-1 Brief description: CCU has to monitor and judge CHAdeMO discrete signalling lines for wakeup-request. The event will be evaluated for validity and CCU will wake up in case of validly detect wake-up-event
  • 1.1.28.11.1.0-2 Preconditions: CHAdeMO communication is not active
  • 1.1.28.11.1.0-3 Trigger: CHAdeMO signalling lines
  • 1.1.28.11.1.0-4 Input data: voltage level von CHAdeMO line
  • 1.1.28.11.1.0-5 Description of behaviour: in case of valid wake-up the CHAdeMO communication will be started
  • 1.1.28.11.1.0-6 Timing Requirements: less than 500ms after apply of wakeup-condition
  • 1.1.28.11.1.0-7 Output data: none
  • 1.1.28.11.1.0-8 Postconditions: CHAdeMO communication is ongoing
  • 1.1.28.11.1.0-9 Descriptions of exceptions: none
  • 1.1.28.11.1.0-10 Dependencies and interactions: none
