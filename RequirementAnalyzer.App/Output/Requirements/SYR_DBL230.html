SYR_DBL230


  • 7.6.1.23 SYR_DBL230: pointer based calibration - read calibration data
  • 7.6.1.23.1 Description
  • 7.6.1.23.1.0-1 Brief description: Calibration tool reads calibration data using refPageAddress. For definition and terms see also: https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.03_SystemArchitecturalDesign/ENG.03_SystemArchitecturalDesignSpecification/AdditionalDocuments/Calibration/bmw_ccu_calibration_architecture.pptx
  • 7.6.1.23.1.0-2 Preconditions: XCP session established
  • 7.6.1.23.1.0-3 Trigger: XCP read command from calibration tool containing refPageAddress
  • 7.6.1.23.1.0-4 Input data: refPageAddress from calibration tool for reading data
  • 7.6.1.23.1.0-5 Description of behaviour: (1) XCP slave searches for slotIndex in OriginPointerTable using refPageAddress. (2) XCP slave reads data using pointer in onlinePointerTable using slotIndex Data taken either from reference page in non-volatile memory (ROM) or working page in volatile memory (ROM).
  • 7.6.1.23.1.0-6 Timing Requirements: command execution shall occur at low priority.
  • 7.6.1.23.1.0-7 Output data:Data read during step (2) above.
  • 7.6.1.23.1.0-8 Postconditions: None
  • 7.6.1.23.1.0-9 Descriptions of exceptions: Use case always succeeds.
