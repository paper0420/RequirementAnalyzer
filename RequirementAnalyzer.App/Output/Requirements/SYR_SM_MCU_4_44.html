  • 7.10.1.1.111.1.0-1 Brief description:The margin read assesses the quality of the flash memory. If the margin read fails, there is a certain probability that the entire flash array is going to have a problem with data retention. In case of ECC error in the flash, we recommend assessing the status of the flash array running a Margin Read. You don't need to run the margin read just after each ECC event. The requirement is to run the MBIST within few trip cycle when you have time to run it. The procedure of margin lead is as follows "37.4.7.2 Margin Read" The MEMU can be used to store the address of the location reporting the error event. "37.3.2.17 User Test 0 register Margin Read Enable(MRE)"
