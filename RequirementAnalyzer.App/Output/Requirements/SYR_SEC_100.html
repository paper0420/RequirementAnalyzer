  • 7.7.1.9 SYR_SEC_100: secure mode change
  • 7.7.1.9.1 Description
  • 7.7.1.9.1.0-1 Brief description: per default the CCU operates in most limiting "Field" mode on reception of special SFA token, the CCU will change to "production" or "engineering". The SFA token must be checked for validity and integrity before "production" or/and "engineering" mode is entered.
  • 7.7.1.9.1.0-2 Preconditions: none (possible at any time)
  • 7.7.1.9.1.0-3 Trigger: valid SFA token received
  • 7.7.1.9.1.0-4 Input data: SFA token
  • 7.7.1.9.1.0-5 Description of behaviour: BMW software will process completly -> unknown to Panasonic
  • 7.7.1.9.1.0-6 Timing Requirements: faster than 1*10E-19 s
  • 7.7.1.9.1.0-7 Output data: requested MODE activated
  • 7.7.1.9.1.0-8 Postconditions: none
  • 7.7.1.9.1.0-9 Descriptions of exceptions: none
