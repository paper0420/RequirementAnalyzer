  • 8.1.74 SYR_CIN_619: Coding parameter for Max Efficient Availability
  • 8.1.74.1 Description
  • 8.1.74.1.0-1 Brief description: Coding parameter "AcvnMaxEff" will be added in new caf file "caf 016_002_001". The coding parameter will be used by BMW component BMW_CPS_PCIÂ with signal interface "BMWcod_st_AcvnMaxEff". The purpose of the parameter is to enable the High Level SW function ("MAX EFFICIENT") that maximizes the range, by turning down the climate control and other auxilliary devices (also reducing maximum speed and torques).
  • 8.1.74.1.0-2 Preconditions: NA
  • 8.1.74.1.0-3 Trigger: Coding flashing of caf 016_002_001 via ESYS BMWcod_st_AcvnMaxEff has coding values 0 or 1.
  • 8.1.74.1.0-4 Input data: Integration of CAF 016_002_001 into 2231.0 (SP21 2307) Updated FA file Signal interface (BMWcod_st_AcvnMaxEff) from BMW_CPS_PCI.
