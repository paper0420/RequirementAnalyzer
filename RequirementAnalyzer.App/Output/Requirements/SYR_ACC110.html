  • 2.1.12 SYR_ACC110: Component protection -Detect and handle undervoltage on DC HV
  • 2.1.12.1 Description
  • 2.1.12.1.0-1 Brief description:
  • 2.1.12.1.0-2 Detect and handle under voltage on the DC side, set error status bit "Abschaltung Unterspannung DC-HV"
  • 2.1.12.1.0-3 Preconditions:
  • 2.1.12.1.0-4 LDK commands the target operation state "charge".
  • 2.1.12.1.0-5 Trigger:
  • 2.1.12.1.0-6 HVDC_Voltage < Threshold_Undervoltage_HVDC
  • 2.1.12.1.0-7 Input data:
  • 2.1.12.1.0-8 LT4_U_Sig
  • 2.1.12.1.0-9 Description of behaviour:
  • 2.1.12.1.0-10 * Detect under voltage on the DC output side: HVDC_Out_Volt < Threshold_Undervoltage_DC * Error persists for more than Threshold_Undervoltage_DC_time * Register the respective DTC and set bit "Abschaltung Unters-pannung DC-HV" in the error status word implementation hint DTC Codes: (Bauteilschutz - DC Unterspannung) Modul 1 0x318E7 Modul 2 0x318ED Modul 3 0x318F5
  • 2.1.12.1.0-11 Timing Requirements:
  • 2.1.12.1.0-12 HVDC_Volt < Threshold_Undervoltage_DC and t_error > Threshold_Undervoltage_DC_time the error persists during a configurable time period (eg., 5 seconds).
  • 2.1.12.1.0-13 Output Data:
  • 2.1.12.1.0-14 "Abschaltung Unterspannung DC" Statuswort 0--- ---- ---- --1- Unterspannung AC
  • 2.1.12.1.0-15 Postconditions:
  • 2.1.12.1.0-16 Descriptions of exceptions:
  • 2.1.12.1.0-17 If an error was found "Abschaltung Unterspannung DC-HV" bit shall be set in the error status word.
