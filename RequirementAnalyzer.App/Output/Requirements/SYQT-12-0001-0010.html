#SYQT-12-0001-0010#

[Charging],[14V_DCDC],[DCDC_Control]
Test Objective: Operation confirmation when the LVDC input voltage is out of range · Error memory registration · When the output current is 0 A

Requirements

  • 14506 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Zustandsmanagement" serves for managing the operating states of the "DC/DC" function: "Init", "Standby", "Buck", "Error" and "Crash". The respective active states are communicated to other functions. The respective operating mode adopted by the subfunction "Zustandsmanagement" corresponds with the operating mode of the "DC/DC" function.
  • 14508 -
    [Not Changed][accepted][SYQT (ENG.10)][Information]
  • 14528 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Gleichspannungswandlung" shall prevent the output transfer between high-voltage network and low-voltage network if the "DC/DC" converter function is in the "Error" operating mode.
  • 14558 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall be able to detect system errors of the DC/DC function.
  • 14559 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall be able to detect sensor errors (short to positive, short to battery voltage (Vbat), open line) of the DC/DC function.
  • 14560 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall make an error memory entry if there is a DC/DC system error.
  • 18174 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall ensure that a failure of the DC/DC converter always leads to a corresponding error memory entry.
  • 18175 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall ensure that an error memory entry is also set if the DC/DC converter does not supply any output power even though this should be provided.
  • 18176 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall set an error memory entry if a DC/DC sensor error exists.
  • 18177 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall set a primary error memory entry if the minimum or maximum operating limits of the DC/DC converter are violated. (e.g. "p_DCDC_DERAT_MIN_U_HV" or "p_DCDC_DERAT_MIN_U_LV" are undershot)
  • 11464 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) connect the development object with resistive load. b) activate and operate the development object at the constant-voltage output mode with the output voltage being a set value Uz0 in the range of output voltage of the development object.
  • 11465 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) adjust the input voltage of the development object such that it varies within ± 15% of the rated value. b) adjust the output current such that it varies within the range between no load to rated current. b) measure actual output voltage Uz.
  • 17648 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) output voltage tolerance must be not more than ± 1% calculated per formula: Of which, ξU -- Output voltage tolerance; Uz -- Actual output voltage (V); Uz0 -- Set output voltage (V).
