#SYQT-05-0001-0039#

[Charging],[AC_Charging],[AC_Charging_Control]
Test Objective: Confirmation that the HVDC output current is limited (three phases) *) Requirement is "not exceed 51.5 A

Requirements

  • 23667 -
    [Not Changed][partly accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    While AC charging, the CCU shall limit the output current on HVDC as follows: - 16,5A for 3,5KW charger - 32,5A for 7KW charger - 51,5A for 11KW charger - 51,5A for 15KW charger - 69,5A for 22KW charger (Calibrateable) Note: The current limit is depends on cables rated current value
  • 17644 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) set the output current limitation value of development object as specified by BMW. b) activate the development object.
  • 17645 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) operate the development object at the constant-voltage state b) adjust the load resistance such that the output current of the development object increases and gradually reaches the set value for current limitation as specified by BMW.
  • 17646 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) during operation, the development object must automatically limit the increment of its output current when the output current attains the current limitation setting value as specified by BMW.
