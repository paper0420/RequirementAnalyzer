  • 7.6.1.28 SYR_DBL_262 : Provide NVM Proxy Data in the Init Context
  • 7.6.1.28.1 Description
  • 7.6.1.28.1.0-1 Brief description: BMW requires the data from the last driving cycle to be made availabe in the Initialization Context for instance for the below listed BMW SWCs. BMW SWCs : BMW_MOD_IlfUii_AllIns BMW_MOD_IlfStai BMW_MOD_FanStc BMW_MOD_WtrPmpStc BMW_MOD_RngCalc BMW_MOD_RngCalcEd BMW_MOD_RngCalcVm BMW_MOD_RngDisp Based on the Read NVM data from the last driving Cycle ,BMW SWCs would be able to use either the NVM value from last driving cylce or an alternate calibration data. [optional]In case the Calibration data is used , this calibration data should also be written back to NvProxy in the same initialization runnable context.
