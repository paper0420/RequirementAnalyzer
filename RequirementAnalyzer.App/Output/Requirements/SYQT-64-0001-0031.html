#SYQT-64-0001-0031#

[Thermal_Management],[AKKS],[Error_Handling]
Test Objective: Stop Test Abort abnormally without detecting blockade with closed.

Requirements

  • 25177 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The status "adopt stepper motor position" is set if a stop test CLOSED is active and the flap is blocked in the lower stop OR if after wake-up the actuator "actual programming" is the same as the "target programming" AND the actual position is outside the normal travel path.
  • 25178 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    In this state, the actual position shall be set to an applicable central position.
  • 25227 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The diagnostic function for monitoring the valid range shall detect an undershooting of the valid applicable signal range (lower stop not detected, can only be tested during stop test CLOSED)
  • 25231 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    Carried out applicatively by each error, it shall be possible to set the target position to an applicative default position.
  • 25232 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the AKKS is in default position it shall be possible to deactivate individual diagnostics.
  • 25239 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If this value is reached without blockade within the traversing range the stop test CLOSED is failed.
  • 25240 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the stop test CLOSED failed, a debouncing shall start (applicable number of attempts).
  • 25241 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the blockade cannot be debounced an error message (DFC) shall be generated. (see Table 1)
  • 25247 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    Requirement for the start of stop test OPEN is that stop test CLOSED is already passed successfully.
