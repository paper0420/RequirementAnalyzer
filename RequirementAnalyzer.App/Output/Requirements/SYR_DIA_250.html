  • 7.5.1.25 SYR_DIA_250: activation and processing of montage-modus
  • 7.5.1.25.1 Description
  • 7.5.1.25.1.0-1 Brief description: on demand the CCU can be set to "Montage-Modus" via diagnosis request. The "Montage-Modus" will: - stop / disable water-pumps - stop / disable DC/DC converter - stop / disable E-ventilator -stop / disable
  • 7.5.1.25.1.0-2 Preconditions: none
  • 7.5.1.25.1.0-3 Trigger: request to activate "Montage-Modus" via diagnosis-tester
  • 7.5.1.25.1.0-4 Input data: requested state of "Montage-Modus" ON/OFF
  • 7.5.1.25.1.0-5 Description of behaviour: CCU will enable/disable functions as described before
  • 7.5.1.25.1.0-6 Timing Requirements: none, CCU has to "ack" the processed request via UDS reply
  • 7.5.1.25.1.0-7 Output data: none
  • 7.5.1.25.1.0-8 Postconditions: described features will be deactivated
  • 7.5.1.25.1.0-9 Descriptions of exceptions: none
  • 7.5.1.25.1.0-10 Dependencies and interactions:none

  • 7.5.1.26 SYR_DIA_260: provide diagnostic service to activate/deactivate PLC mirror feature - Obsolete
  • 7.5.1.26.1 Description
  • 7.5.1.26.1.0-1 Brief description: SYR became obsolete after cancellation of feature through CR-822 ------------------------------------------------------------------------------------------ provide diagnostic service to activate/deactivate PLC Mirror Feature. This service shall be used for debugging purpose in Feild and development phase.
  • 7.5.1.26.1.0-2 Preconditions: SPI interface between QCA chip and Controller shall be in operation and Ethernet interface shall be preintialised.
  • 7.5.1.26.1.0-3 Trigger: Diagnostic tested
  • 7.5.1.26.1.0-4 Input data: NA
  • 7.5.1.26.1.0-5 Description of behaviour: Activation request: After activation, PLC traffic (Tx/Rx) shall also be visible on Ethernet Interface of CCU. Deactivation request: CCU shall stop mirror traffic over Ethernet interface. Status: On query from diagnostic tester, CCU shall provide status if it is activated/deactivated or in error.
  • 7.5.1.26.1.0-6 Timing Requirements: none,
  • 7.5.1.26.1.0-7 Output data: mirrored PLC frames on OABR
