SYR_PLC_14


  • 1.1.30.16 SYR_PLC_14: init PLC interface
  • 1.1.30.16.1 Description
  • 1.1.30.16.1.0-1 Brief description: This usecase is part of SYR_PLC_02 and SYR_PLC_03. The Usecase is created to mention the expectation of power up/down seq for plc interface. When the powermanagemnet component request to init PLC interface. note this activity shall already be implemented by SYR_PLC_02 and SYR_PLC_03. it created to capture the init seq for power managemnet.
  • 1.1.30.16.1.0-2 Preconditions: CCU is powerdown mode. no supply provided.
  • 1.1.30.16.1.0-3 Trigger: powermanagement component trigger the startup init.
  • 1.1.30.16.1.0-4 Input data: na
  • 1.1.30.16.1.0-5 Description of behaviour: The expectation is when power management component request for plc interface init, power shall be applied to the plc chipset and reset pin shall not be release to start the bootup seq from chorus. The spi interface to communicate with plc chipset shall be initialised and kept ready to read/write any configuration to internal registors of plcchipset. later this spi interface shall be used to send control service information and data service information.
  • 1.1.30.16.1.0-6 Timing Requirements: as requested by PM component.
  • 1.1.30.16.1.0-7 Output data: na
  • 1.1.30.16.1.0-8 Postconditions: interface is ready to bootup via chorus on bmw swc trigger.
