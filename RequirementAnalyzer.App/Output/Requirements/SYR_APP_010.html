  • 7.4.1.1 SYR_APP_010: startup
  • 7.4.1.1.1 Description
  • 7.4.1.1.1.1 Brief description:
  • 7.4.1.1.1.1.0-1 The main-application will be started: This include the initialisation of complete hardware and operating system. In this mode the CCU is ready to operate its primary functions. The CCU is able to operate all kind of power-management and communication via all external busses. The diagnosis-application is available completely. The main-application will NOT operate any flash-update activities else than “CAF” (Coding Application File) update. This use-case is basically the operation of complete main-application
  • 7.4.1.1.1.1-1 Preconditions:
  • 7.4.1.1.1.1-1.0-1 The device is in one of the following system states: SOC is powered and BootManager decided to start main-application
