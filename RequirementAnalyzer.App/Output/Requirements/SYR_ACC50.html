SYR_ACC50


  • 2.1.5 SYR_ACC50: Function: "Messwertermittlung" / Safety function-measure input HV AC voltage
  • 2.1.5.1 Description
  • 2.1.5.1.0-1 Brief description:
  • 2.1.5.1.0-2 determine the actual root‐mean‐square value of AC voltage between the connected conductor and the neutral conductor of the charger for L1, L2 and L3
  • 2.1.5.1.0-3 Preconditions:
  • 2.1.5.1.0-4 none (always)
  • 2.1.5.1.0-5 Trigger:
  • 2.1.5.1.0-6 start up (always active)
  • 2.1.5.1.0-7 Input data:
  • 2.1.5.1.0-8 E_U_HVAC_L1Phy E_U_HVAC_L2Phy E_U_HVAC_L3Phy
  • 2.1.5.1.0-9 Description of behaviour:
  • 2.1.5.1.0-10 measure the rms voltage value from the ADC for the different phases over one monitoring cycle. Distribute over the respective signals. The required accuracy should be respected. The function shall be implemented with ASIL A Implementation hint. Signal names for distribution of the values: * send E_U_HVAC_L1Phy to HVAC_U_L1Sig * send E_U_HVAC_L3Phy to HVAC_U_L2Sig * send E_U_HVAC_L3Phy to HVAC_U_L3Sig accuracy: 0A < E_U_HVAC_Lx<300V: +/‐ 10V conversion formula: HVAC_Lx_Volt = ( ADC_HVAC_Lx_Volt * 4.096V / 2^12) * ( pi/ 2*SQRT(2) ) * ( 1 / 0.00704225352112676) * ( 1 / ATT) *(1/alpha) Filter, Debounce : moving average 8 times Safe State : Respective Qualifier set to not valid.
  • 2.1.5.1.0-11 Timing Requirements:
  • 2.1.5.1.0-12 monitoring cycle 10ms FTT: 500ms
  • 2.1.5.1.0-13 Output Data:
  • 2.1.5.1.0-14 HVAC_U_L1Sig HVAC_U_L2Sig HVAC_U_L3Sig
  • 2.1.5.1.0-15 Postconditions:
  • 2.1.5.1.0-16 none
  • 2.1.5.1.0-17 Descriptions of exceptions:
  • 2.1.5.1.0-18 Safe state:respective qualifier of the signal HVAC_U_LxSig is set to no valid
