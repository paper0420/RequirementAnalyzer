SYR_PLC_04


  • 1.1.30.5 SYR_PLC_04: provide functionality for updating external flash of QCA chip
  • 1.1.30.5.1 Description
  • 1.1.30.5.1.0-1 Brief description: Panasonic software in CCU should be able to update the external flash of QCA Chip, the pre-status of FLASH could be EMPTY / OLD Firmware or PIB
  • 1.1.30.5.1.0-2 Preconditions: Flash is blank or has old firmware/ PIB
  • 1.1.30.5.1.0-3 Trigger: Trigger from Application (Diag or LIM-C or Esys)
  • 1.1.30.5.1.0-4 Input data: Host provide QCA firmware and PIB configuration file
  • 1.1.30.5.1.0-5 Description of behaviour: - QCA need to firmware from External flash every time it boots from POR
  • 1.1.30.5.1.0-6 Timing Requirements: ISO15118-3 Timing requirements shall be fulfilled by CCU
  • 1.1.30.5.1.0-7 Output data: Physical Link shall be available.
  • 1.1.30.5.1.0-8 Postconditions: external flash is updated with new firmware and next reboot or QCA should read new firmware version number
  • 1.1.30.5.1.0-9 Descriptions of exceptions:
  • 1.1.30.5.1.0-10 Dependencies and interactions: - HOST should have Firwmare and PIB copy (one set)
