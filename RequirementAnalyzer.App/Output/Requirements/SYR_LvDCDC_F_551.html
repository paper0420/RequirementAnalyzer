  • 6.1.72.1.0-9 When the LvDCDC stops the buck mode due to the KL_30 undervoltage the error state is set accordingly and a DTC is generated. The LvDCDC is in the error operation mode as long as the error condition is present. If the undervoltage condition is finished, the LvDCDC is in standby operation mode. The buck operation mode cannot be started without HVPM.
