#SYQT-19-0001-0013#

[Charging],[Component_Protection],[DC_Overvoltage]
Test Objective: Confirmation of overvoltage detection of HVDC output voltage (China requirement)

Requirements

  • 11401 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) connect a DC electronic load to the output end of the development object. b) activate the development object at input voltage (± 15% of the rated input voltage) and input frequency (50 Hz ± 2%) c) operate the development object at the constant-current state.
  • 17628 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) adjust the electronic load voltage such that it gradually attains the DC overvoltage protection value of development object as specified by BMW.
  • 11402 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) the development object must shut down the output and send a bus message when the input voltage is more than the overvoltage protection value. b) the development object must be able to restore output voltage and output current automatically when the malfunction is eliminated.
