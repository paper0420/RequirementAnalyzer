SYR_COA_081


  • 7.9.1.14 SYR_COA_081: Calibratable Activation of End to End protection
  • 7.9.1.14.1 Description
  • 7.9.1.14.1.0-1 Brief description: This UC gives brief description of the E2E protection with Calibration facility for the signals received by ComAdapter.
  • 7.9.1.14.1.0-2 Preconditions: RUNLEVEL>=30
  • 7.9.1.14.1.0-3 Trigger: When the signal is received by Com Adapter and the calibration access for the signal is configured as ReadWrite then the signal End to End protection can be switched between Readwrite and NotAccessible states.
  • 7.9.1.14.1.0-4 Input data: Valid COM signal
  • 7.9.1.14.1.0-5 Description of behaviour: When the Calibration Access is configured as "ReadWrite" then the calibration parameter read and write access is enabled. When the Calibration Access is configured as "ReadOnly" then the calibration parameter read access is enabled. ComAdapter shall use the E2E protection wrapper mechanism to send and receive the data elements.
  • 7.9.1.14.1.0-6 Timing Requirements: (Message cycle time + tolerance) ms
  • 7.9.1.14.1.0-7 Output data: The COM signal End to End protection is Enabled when Calibration access is activated The COM signal End to End protection is Disabled when Calibration access is deactivated
