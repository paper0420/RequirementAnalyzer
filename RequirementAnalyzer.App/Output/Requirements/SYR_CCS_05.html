  • 1.1.31.8 SYR_CCS_05: provide CCS protocol support for ISO15118 AC message set
  • 1.1.31.8.1 Description
  • 1.1.31.8.1.0-1 Brief description: Panasnoic software shall provide ISO15118 AC message support. PLC stack configuration shall be done by Panasonic, The PANA SW shall communicate with charging station, collect information and provide to BMW SW or Charging station as per message set
  • 1.1.31.8.1.0-2 Preconditions: EVSE with CCS/ISO15118 is connected to CCU
  • 1.1.31.8.1.0-3 Trigger: EVSE applies valid duty-cycle on Pilot-Line
  • 1.1.31.8.1.0-4 Input data: analog values of Pilot-Line (duty-cycle/voltage level), proximity-line voltage level, PLC-communication of Pilot-Line
  • 1.1.31.8.1.0-5 Description of behaviour: CCU will communicate with EVSE like described in ISO-15118 Please refer seq https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.02_SystemRequirementsAnalysis/ENG.02_SystemRequirementSpecification/AdditionalDocuments/Charging_Interface/CCS/Sequence/LH_CCS_ISO15118_AC_EIM_Sequence.pdf https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.02_SystemRequirementsAnalysis/ENG.02_SystemRequirementSpecification/AdditionalDocuments/Charging_Interface/CCS/Sequence/LH_CCS_ISO15118_AC_PnC_Sequence.pdf
  • 1.1.31.8.1.0-6 Timing Requirements: define by ISO15118
  • 1.1.31.8.1.0-7 Output data: defined by ISO15118
  • 1.1.31.8.1.0-8 Postconditions: trigger from bmw SWC and Datalink must be established
  • 1.1.31.8.1.0-9 Descriptions of exceptions: none
  • 1.1.31.8.1.0-10 Dependencies and interactions: none
