#SYQT-12-0002-0015#

[Charging],[14V_DCDC],[DCDC_Control]
Test Objective: Operation check at LVDC overcurrent · 0 A for output current · Error memory registration

Requirements

  • 14533 -
    [Not Changed][accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    The subfunction "Gleichspannungswandlung" shall trigger the self-protection mechanism of the DC/DC converter if the HV currents exceed the value "p_DCDC_HV_CURRENT_MAX_ALLOWED" (depends on hardware) and can damage the DC/DC converter.
  • 18153 -
    [Not Changed][accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    If the overcurrent self-protection successively triggers for a number of "p_DCDC_REPEAT_CURRENT_HIGH_CHECK", the converter shall permanently deactivate and change to the state "Error".
  • 14534 -
    [Not Changed][accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    The subfunction "Gleichspannungswandlung" shall trigger the DC/DC converter self-protection mechanism if the LV currents exceed the value "p_DCDC_LV_CURRENT_MAX_ALLOWED" (depends on the hardware) and can damage the DC/DC converter.
  • 18157 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Gleichspannungswandlung" shall execute an error memory entry for a deactivation that occurs due to an overcurrent event.
  • 14558 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall be able to detect system errors of the DC/DC function.
  • 14560 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall make an error memory entry if there is a DC/DC system error.
  • 18174 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall ensure that a failure of the DC/DC converter always leads to a corresponding error memory entry.
  • 18175 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall ensure that an error memory entry is also set if the DC/DC converter does not supply any output power even though this should be provided.
  • 18176 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Fehlermanagement" shall set an error memory entry if a DC/DC sensor error exists.
