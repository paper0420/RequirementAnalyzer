#SYQT-57-0001-0002#

[Device_Control],[S5_Sensor],[S5_Sensor]
Test Objective: When the temperature is outside the measurement range, detect the error and confirm that the DTC code is registered.

Requirements

  • 25075 -
    [Not Changed][accepted][SYQT (ENG.10)Review][Requirement]
    All information generated by the diagnostic and evaluation functions shall be readable.
  • 25079 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The diagnostic function shall monitor the temperature sensor-coolant radiator outlet signal by means of an out-of-range check for physically plausible values within the electrical limits.
  • 25080 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The threshold values for the out-of-range check shall be implemented applicably.
  • 25083 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    After debouncing, the relevant error bit shall be set to "TRUE."
  • 25087 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The diagnostic function shall monitor the signal of the temperature sensor coolant radiator outlet for plausible values within the physical limits of the out-of-range check.
  • 25089 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The diagnostic function shall generate status information about the current signal quality of the temperature sensor coolant radiator outlet.
  • 25101 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - The respective error bit shall be initialized with "FALSE."
  • 25102 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - The threshold values for the detection of each type of electrical error shall be implemented applicably.
  • 25104 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - Debouncing shall be implemented applicably for each type of electrical error.
  • 25105 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - After debouncing, the relevant error bit shall be set to "True."
