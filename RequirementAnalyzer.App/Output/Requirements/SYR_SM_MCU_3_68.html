  • 7.10.1.1.48.1.0-1 Brief description: Enable CRC function of STCU2. The CRC function is enabled by setting STCU_CFG [CRCEN] to enable and setting the value used for calculation of CRC to the STCU_CRCE register. Check the CRCS / CRCSSW bit in the STCU_ERR_FM register to see if a CRC error has occurred.
