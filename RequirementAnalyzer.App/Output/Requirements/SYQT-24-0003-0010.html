#SYQT-24-0003-0010#

[Charging],[Component_Protection],[AC_Undervoltage]
Test Objective: Confirm that the charging operation stops at AC low voltage detection (China specification)

Requirements

  • 11406 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) activate and operate the development object at input voltage (± 15% of the rated input voltage) and input frequency (50 Hz ± 2%).
  • 17630 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) adjust the input voltage of the development object in order that it is lower than the protection value of input under-voltage as specified by BMW.
  • 11407 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) the development object must shut down the output and send a bus message when the input voltage is less than the undervoltage protection value. b) the development object must be able to restore output voltage and output current automatically when the malfunction is eliminated.
