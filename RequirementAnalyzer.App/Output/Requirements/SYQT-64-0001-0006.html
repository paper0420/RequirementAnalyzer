#SYQT-64-0001-0006#

[Thermal_Management],[AKKS],[basic action]
Test Objective: During Stop Test Open, Stop Test Open will not stop even if the vehicle speed exceeds 90 km / h.

Requirements

  • 25218 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Information that lower stop was learned shall remain active as long as the control unit is still awake (= Control unit awake cycle). (volatile memory)
  • 25244 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The result of the lower stop test has been detected shall remain stored for one control-unit wake-cycle in order to avoid an unnecessary stop learning process.
  • 26945 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the upper speed limit is exceeded it must be ensured that the function has no deadlock in the stop test open.
  • 26947 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The result of the lower stop test has been detected shall remain stored for one control-unit wake-cycle in order to avoid an unnecessary stop learning process.
