SYR_DSP_010


  • 7.4.1.8 SYR_DSP_010: normal application
  • 7.4.1.8.1 Description
  • 7.4.1.8.1.1 Brief description:
  • 7.4.1.8.1.1.0-1 The DSP is responsible for DC/DC conversion management. The “normal application” means the nominal execution of this task. While this mode, data is exchanged with the “main application” on “System CPU”. The “main application” is responsible for power-management of the DSPs by switching on/off power supply of DSP. If SW-update is requested, the “bootloader “of “system cpu” will request the DSP to start its “flash bootloader” via SPI. In this case the “normal application” of DSP is stopped and “flash bootloader” is started.
  • 7.4.1.8.1.1-1 Preconditions:
  • 7.4.1.8.1.1-1.0-1 The device is in one of the following system states: Flash of DSP contains valid application-program DSP is powered
