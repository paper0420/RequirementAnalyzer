SYR_FIM_010


  • 7.5.1.55 SYR_FIM_010: handle set of FIDs for FIM
  • 7.5.1.55.1 Description
  • 7.5.1.55.1.0-1 Brief description: This UC will handle state of all possible FIDs. On request it will set properties of referenced FID. On request it will supply snap-shot information about current state of one/all FIDs. A change of FID status is judged as event and will be forewarded to attached information-sink
  • 7.5.1.55.1.0-2 Preconditions: BSW / DEM running
  • 7.5.1.55.1.0-3 Trigger: status change of DEM
  • 7.5.1.55.1.0-4 Input data: referring FID / DTC
  • 7.5.1.55.1.0-5 Description of behaviour: DTC status will change affected FID, filters will be applied to FIT and on demand notification will be issued
  • 7.5.1.55.1.0-6 Timing Requirements: none
  • 7.5.1.55.1.0-7 Output data: state of FID
  • 7.5.1.55.1.0-8 Postconditions: none
