#SYQT-64-0001-0047#

[Thermal_Management],[AKKS],[Error_Handling]
Test Objective: Detect abnormality by deviation detection from current position.

Requirements

  • 25229 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The diagnostic function shall permanently monitor a TARGET-ACTUAL deviation of the position.
  • 25230 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    It shall be possible to debounce the error memories in a suitable manner after consultation (event debouncing or time debouncing).
  • 26957 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    A function shall be implemented to detect a position deviation between the target position and the actual value position.
  • 26958 -
    [Not Changed][partly accepted][SWQT (ENG.8)][Requirement]
    The detection shall be implemented in alle vehicle condition (ST_CON_VEH)
  • 26959 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The position deviation detected shall not be activated at the initialization phase. This requirement shall be applicatable.
  • 26960 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The position deviation error shall be detected if the absolute difference between target and actual value is larger than an applicatable threshold for an applicatable time A and a movement request is active for an applicatable time B.
