  • 1.1.3.2 SYR_CPL31: Lock request received (Ansteuerung_Ladesteckerverriegelung = 2)
  • 1.1.3.2.1 Description
  • 1.1.3.2.1.0-1 Brief description:
  • 1.1.3.2.1.0-2 When a lock request is received, the component shall follow the request by configuring the H-Bridge for a locking actuation. Pana_Actuation_status will be 0 (locking) for the entire duration of the actuation. When the actuation is finished, Pana_Actuation_status shall be 1 (locked) if there was no error detected during the actuation and the actuation time has expired.
