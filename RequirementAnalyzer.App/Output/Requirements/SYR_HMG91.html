  • 3.1.27 SYR_HMG91: DTC 0x224100 "Cooling-pump HVB - Degradation due to Overtemperature"
  • 3.1.27.1 Description
  • 3.1.27.1.0-1 Brief description: the CCU has to be able to detect the failure "Degradation due to Overtemperature" of the Waterpump EWP22 (Cooling-pump of the High-Voltage Battery)
  • 3.1.27.1.0-2 Preconditions:Kl15 ON and LIN Communication between the Waterpump EWP22 and the LIN-Master CCU is correctly working
  • 3.1.27.1.0-3 Trigger: ST_EWP22_LIN.ERR_ST_OTMP_EWP22_LIN=0x02;
  • 3.1.27.1.0-4 Input data: LIN Signal ERR_ST_OTMP_EWP22_LIN
  • 3.1.27.1.0-5 Description of behaviour: The Waterpump EWP22 will signalize to the CCU via LIN the presence of the failure "Degradation due to Overtemperature". The failure will be detected when the signal ERR_ST_OTMP_EWP22_LIN=0x02. As soon as this error condition is detected (after the expiration of a debouncing time whose value has still to be provided by BMW) an entry with Code 0x224100 will be written in the Error Memory

  • 3.1.27.1.0-7 Output data: the DTC with code 0x224100
