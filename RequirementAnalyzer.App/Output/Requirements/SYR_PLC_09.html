  • 1.1.30.10 SYR_PLC_09: provide service to enable/disable Random Traffic on PLC interface
  • 1.1.30.10.1 Description
  • 1.1.30.10.1.0-1 Brief description: Functionality shall be provide to turn on/off random traffic on the plc interface. This traffic will be used for EMC and Power density spectrum measurement at factory. To calibrated the CCU device with car harness this function is must.
  • 1.1.30.10.1.0-2 Preconditions: physical link on PLC interface shall be established
  • 1.1.30.10.1.0-3 Trigger: diag job
  • 1.1.30.10.1.0-4 Input data: enable/disable value via Diag job
  • 1.1.30.10.1.0-5 Description of behaviour: default it shall be disabled. on request it shall enable, this will allow us to see extra random traffic on plc interface.
  • 1.1.30.10.1.0-6 Timing Requirements: standard diagnostic timing shall be realised.
  • 1.1.30.10.1.0-7 Output data: extra traffic on plc interface
  • 1.1.30.10.1.0-8 Postconditions: When Enable we shall see MME frames send out of CCU atleast at 20% max. data rate. Tx Frame Detail: Destination MAC address FF:FF:FF:FF:FF:FF (broadcast address) Source MAC address (MAC of ST Chorus) VLAN (optional) Ethertype (0x88E1) Body : 01010101....(length Max. 1014) or any random values
  • 1.1.30.10.1.0-9 Descriptions of exceptions: NA
  • 1.1.30.10.1.0-10 Dependencies and interactions: Physical Link shall be available before enabling
