SYR_CCS_03


  • 1.1.31.5 SYR_CCS_03: provide CCS protocol support for DIN70121 DC message set
  • 1.1.31.5.1 Description
  • 1.1.31.5.1.0-1 Brief description: Panasonic software shall provide DIN70121 message support. The DIN70121 Parameter confuguration shall be done. Panasonic software shall Communication with Charging Station, Collecting information and providing to BMW SW or Charging station as per message set. The BMW SW will control the communication with the Charging station.
  • 1.1.31.5.1.0-2 Preconditions: Dlink is established and IP is assigned
  • 1.1.31.5.1.0-3 Trigger: LIM_C shall provide trigger start DIN70121 seq
  • 1.1.31.5.1.0-4 Input data: LIM_C shall provide information need for creating request message
  • 1.1.31.5.1.0-5 Description of behaviour: see DIN70121 Seq. The communication with charging station shall be accordance with seq https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.02_SystemRequirementsAnalysis/ENG.02_SystemRequirementSpecification/AdditionalDocuments/Charging_Interface/CCS/Sequence/LH_CCS_DIN70121_Sequence.pdf
  • 1.1.31.5.1.0-6 Timing Requirements: shall as requested in DIN70121 standard
  • 1.1.31.5.1.0-7 Output data: PLC frames to Charging station (external)
  • 1.1.31.5.1.0-8 Postconditions: trigger from bmw SWC and Datalink must be established
  • 1.1.31.5.1.0-9 Descriptions of exceptions: none
  • 1.1.31.5.1.0-10 Dependencies and interactions: none
