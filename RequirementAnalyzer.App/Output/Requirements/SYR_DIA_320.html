SYR_DIA_320


  • 7.5.1.32 SYR_DIA_320: collect Reset DTC environmental data
  • 7.5.1.32.1 Description
  • 7.5.1.32.1.0-1 Brief description: Environmental data for the DTC (ID: 0xBA0023 = Sammelfehler Software) has to be collected during normal CPU operation. The collected data has to be provided in a RAM area that is retained from operation cycle i to operation cycle i+1 with a CPU reset between cycle i and cylce i + 1. Collection occurs either periodically or event driven (preferred) based on collected data. Baisc guideline: CPU load shall be as small as possible.
  • 7.5.1.32.1.0-2 Preconditions: Software is running
  • 7.5.1.32.1.0-3 Trigger: various events - see details of environmental data defined in https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.02_SystemRequirementsAnalysis/ENG.02_SystemRequirementSpecification/AdditionalDocuments/Diag/sammelfehler_sw_i400.xlsx
  • 7.5.1.32.1.0-4 Input data: see above document
  • 7.5.1.32.1.0-5 Description of behaviour: This is continously running or triffered action.
  • 7.5.1.32.1.0-6 Timing Requirements: none.
