  • 7.7.1.18 SYR_SEC_190: HSM Error Logging and Reporting
  • 7.7.1.18.1 Description
  • 7.7.1.18.1.0-1 Brief description: The HSM errors shall be logged in all HSM user SW modules. The HSM errors shall be stored in HSM Dflash. Incase of failure to retrive the HSM error log from HSM's Dflash, a default failure value shall be reported to indicate this state. The Error logging shall follow the format in: "HSM Error Log". Please refer to the sheet "catalog" in the file https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.02_SystemRequirementsAnalysis/ENG.02_SystemRequirementSpecification/AdditionalDocuments/Documentation/Security/SysRequirementsForSecurityEpics.xlsx
