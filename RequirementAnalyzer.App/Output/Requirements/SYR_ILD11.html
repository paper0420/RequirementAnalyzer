  • 1.1.2.2 SYR_ILD11: CCU basic Mode management of Intelligent Ladedose
  • 1.1.2.2.1 Description
  • 1.1.2.2.1.0-1 Brief description: CCU master has to control wakeup and sleep modes of iLD slave. iLD will be in wakeup mode when AE CAN is active otherwise iLD will be in sleep mode. When CCU is going to sleep mode iLD will be changed to sleep mode.
  • 1.1.2.2.1.0-2 Preconditions:
  • 1.1.2.2.1.0-3 Trigger:
  • 1.1.2.2.1.0-4 Input data: 1. CCU wake up reason is provided as input 2. CCU sleep mode is enabled
  • 1.1.2.2.1.0-5 Description of behaviour: CCU is directly connected to T30 terminal (BAT+) and iLD is connected to T30F switched terminal. CCU will be awake when vehicle bus is active. During sleep, do not stop all messages at a time. Initially stop NM messages and stop other messages with delay.
  • 1.1.2.2.1.0-6 Timing Requirements:
  • 1.1.2.2.1.0-7 Output data: 1. iLD Wake up is enabled 2. iLD sleep mode is enabled.
