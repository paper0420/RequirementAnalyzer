SYR_CIN_390


  • 8.1.43 SYR_CIN_390: Add coding parameter for BAC CDC Module
  • 8.1.43.1 Description
  • 8.1.43.1.0-1 Brief description: Integration of coding parameters for the BAC CDC feature. This system requirement is valid for I20/G26
  • 8.1.43.1.0-2 Preconditions: CCU in flash mode
  • 8.1.43.1.0-3 Trigger: coding CCU
  • 8.1.43.1.0-4 Input data: CAF file, FA file
  • 8.1.43.1.0-5 Description of behaviour: 4 coding parameters for CDC feature (CdcClientEnabled, CdcClientMaxTransmissionRate, CdcClientMaxTransmissionSize and CdcClientTransmissionCycleTime) have to be configured in the BAC Coding Module. BAC Coding module has then to be connected to CDC client, in order to allow CDC coding parameter to be used by CDC client. Acceptance : The coding parameter shall be accessible to CDC client
  • 8.1.43.1.0-6 Timing Requirements: NA
  • 8.1.43.1.0-7 Output data: BMW coding parameter signal is set as per FA file
  • 8.1.43.1.0-8 Postconditions: The coding parameter shall be accessible to BAC CDC Client
  • 8.1.43.1.0-9 Descriptions of exceptions: NA
  • 8.1.43.1.0-10 Dependencies and interactions: NA
