#SYQT-17-0001-0033#

[Charging],[Component_Protection],[Derating]
Test Objective: Cofirm that HVDC output power is not derated when HVDC output starts from the high temperature state (where the derating power is higher than the actual output power). (Variant:3.7kW with part-load charging)

Requirements

  • 18173 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    The subfunction "Fehlermanagement" shall read in information on the existing DC/DC converter error state for the logical signal "Fehlerstatus_DCDC-Wandler_Ist".
  • 18386 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If the temperature up to which the charging module shall not derate is exceeded, the derating power shall correspond to the maximum output value that the hardware can still provide.
