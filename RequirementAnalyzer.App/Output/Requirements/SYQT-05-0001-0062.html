#SYQT-05-0001-0062#

[Charging],[AC_Charging],[AC_Charging_Control]
Test Objective: Confirmation that the HVDC output current is limited (three phases)

Requirements

  • 23667 -
    [Not Changed][partly accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    While AC charging, the CCU shall limit the output current on HVDC as follows: - 16,5A for 3,5KW charger - 32,5A for 7KW charger - 51,5A for 11KW charger - 51,5A for 15KW charger - 69,5A for 22KW charger (Calibrateable) Note: The current limit is depends on cables rated current value
