  • 7.7.1.20 SYR_SEC_210: Supplier session after Panasonic EOL
  • 7.7.1.20.1 Description
  • 7.7.1.20.1.0-1 Brief description: Supplier session is enabled by default and get locked at EOL. Change required: Provision to re-enable supplier session even after EOL, if required. Reason for change: There might arise a rare situation where we need to re-activate supplier session. Use cases are not identified. So this provision is required rather than locking it permanently.
  • 7.7.1.20.1.0-2 Preconditions: Supplier Session is locked
  • 7.7.1.20.1.0-3 Trigger: Tester request to unlock the supplier session

  • 7.7.1.20.1.0-8 Postconditions: when condition is satisfied, the supplier session is unlocked.
