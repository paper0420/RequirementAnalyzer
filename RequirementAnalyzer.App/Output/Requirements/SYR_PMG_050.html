SYR_PMG_050


  • 7.2.1.18 SYR_PMG_050: stop Software-componentes
  • 7.2.1.18.1 Description
  • 7.2.1.18.1.0-1 Brief description:
  • 7.2.1.18.1.0-2 This UC is primary intended to start/monitor/force shut down of SW-components. It will consider the dependencies between SW-components to prevent the shutdown of a component which is still needed by other running components. This UC will supervise the shutdown and manage situations if one/some SW-components do not stop after request. If timeout time has elapsed, this UC will push the further shutdown even if some SW-component did not stop as requested.
  • 7.2.1.18.1.0-3 Preconditions: power management decided to lower RUNLEVEL
  • 7.2.1.18.1.0-4 Trigger: RUNLEVEL change
  • 7.2.1.18.1.0-5 Input data: intended RUNLEVEL
  • 7.2.1.18.1.0-6 Description of behaviour: all components which are not needed in new RUNLEVEL are stopped
  • 7.2.1.18.1.0-7 Timing Requirements: none
  • 7.2.1.18.1.0-8 Output data: none
  • 7.2.1.18.1.0-9 Postconditions: component is stopped
  • 7.2.1.18.1.0-10 Descriptions of exceptions: none
  • 7.2.1.18.1.0-11 Dependencies and interactions: none
