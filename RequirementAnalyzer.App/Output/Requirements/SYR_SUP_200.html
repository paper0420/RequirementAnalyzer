SYR_SUP_200


  • 7.4.2.21 SYR_SUP_200: modification of data to be flashed into CCU
  • 7.4.2.21.1 Description
  • 7.4.2.21.1.0-1 Brief description: the CCU will pass meta-data and data to be flashed into iLD. This data needs to be modified on the fly: The CCU has to reduce the size, informed to be transferes and drop the data to be flashed.
  • 7.4.2.21.1.0-2 Preconditions: iLD-Software Update is ongoing
  • 7.4.2.21.1.0-3 Trigger: ESYS intends to update the iLD
  • 7.4.2.21.1.0-4 Input data: "request download" / "transfer data"
  • 7.4.2.21.1.0-5 Description of behaviour: the CCU will reduce the indicated data-size from ESYS before this indicate is send to iLD. The ILD will drop additional data transfered by ESYS before forwarding to iLD
  • 7.4.2.21.1.0-6 Timing Requirements: modification must be done "on the fly" keeping the UDS-timing
  • 7.4.2.21.1.0-7 Output data: modified size of data / modified data to be flashed
