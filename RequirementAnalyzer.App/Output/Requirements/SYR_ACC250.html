  • 2.1.29.1.0-9 * verify current configuration of the switch matrix (SM) * transmit current SM status to the LDK with the required frequency implementation hint send data with the following structure 0: Initial 1: 1 phase charging mode 3: 3 phase charging mode 11: Error, stuck in 1 phase charging mode 13: Error, stuck in 3 phase charging mode 14: Overvoltage, charging stopped 15: Global Error, charging stopped Initially SLE shall always send sm status WITHOUT STUCK ERROR. That means 1. If the initial sm configuration is 1 or 11, SLE shall send 1 2. If the initial sm configuration is 3 or 13, SLE shall send 3 (this allows for a faster detection of the last configuration and allows for a faster charging inizializatuion)
