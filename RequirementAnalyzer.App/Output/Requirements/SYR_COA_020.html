  • 7.9.1.6.1 Description
  • 7.9.1.6.1.0-1 Brief description:
  • 7.9.1.6.1.0-2 This UC gives brief description for the Class 1 signal quality signals having Signal tag and Signal Qualifier. The Internal Qualifier is generated by Com Adapter using Signal Tag. In a Signal Group if multiple Signal Tags are present then Maximum value of Signal tag is considered for Internal Qualifier determination. Signal Qualifier represents the application monitoring status. The Signal Qualifier of the corresponding signal are transmitted over the board network.The internal Qualifier in the software is used by Com Adapter to generate Signal Tags.The signal qualifier in general is represented in 4 bits where 15 shows invalid value and other value shows the application status.
