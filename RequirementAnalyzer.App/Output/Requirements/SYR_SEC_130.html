  • 7.7.1.12 SYR_SEC_130: Moving Production from NVM to Flash memory
  • 7.7.1.12.1 Description
  • 7.7.1.12.1.0-1 Brief description: Some data need to be moved to Flash memory
  • 7.7.1.12.1.0-2 Preconditions: Supplier Session active
  • 7.7.1.12.1.0-3 Trigger: Send read diag service send write diag service
  • 7.7.1.12.1.0-4 Input data: none
  • 7.7.1.12.1.0-5 Description of behaviour: Below data must be moved to Flash memory: 1.SerialNumber 2.ECU ID 3.Programming Date 4.HWEL 5.XCP or XETK For diag domain: Please re-route the diag service to read from the flash memory instead of NVM write command for these data will be disabled as it will be written in Factory. For old units: before updating the units with the new SW having this EPIC, make read and store in a file the values. an INCA profile will be created and using these values, will be written in flash.
