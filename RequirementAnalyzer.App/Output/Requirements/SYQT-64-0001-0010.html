[Thermal_Management],[AKKS],[Basic_Operation]
Test Objective: In PAD mode AKKS2 flap movement is possible

Requirements

  • 25169 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    In the case of undervoltage, it shall be possible to activate/deactivate the function via the application.
  • 25190 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
