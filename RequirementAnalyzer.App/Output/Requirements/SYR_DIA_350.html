SYR_DIA_350


  • 7.5.1.36 SYR_DIA_350: execute Diagnosis 'upload service'
  • 7.5.1.36.1 Description
  • 7.5.1.36.1.0-1 Brief description: This service (Id=0x35) is the opposite to download service. The tester contacts the CCU with a memory address and the amount of data to be transferred from the ECU to the tester.
  • 7.5.1.36.1.0-2 Preconditions: Tester requests upload of data in a extended diagnosis session. See ZEDIS for conditions related to security mode of ECU.
  • 7.5.1.36.1.0-3 Trigger: Command from Tester received
  • 7.5.1.36.1.0-4 Input data: Memory address to read and number of bytes to read
  • 7.5.1.36.1.0-5 Description of behaviour: read data from memory and send UDS frames back to tester containing this data.
  • 7.5.1.36.1.0-6 Timing Requirements: none.
  • 7.5.1.36.1.0-7 Output data: contiguous block of memory data send in one or more UDS frames.
  • 7.5.1.36.1.0-8 Postconditions: none
  • 7.5.1.36.1.0-9 Descriptions of exceptions: none
  • 7.5.1.36.1.0-10 Dependencies and interactions: none
