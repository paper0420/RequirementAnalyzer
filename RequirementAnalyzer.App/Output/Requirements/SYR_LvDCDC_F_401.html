SYR_LvDCDC_F_401


  • 6.1.54 SYR_LvDCDC_F_401: LvDCDC is stopping output current due to minimum permitted high-voltage value
  • 6.1.54.1 Description
  • 6.1.54.1.0-1 Brief description:
  • 6.1.54.1.0-2 The LvDCDC stops the output when the minimum permitted high-voltage value, given by HVPM is reached. The LvDCDC starts the output when the HV input voltage is above.
  • 6.1.54.1.0-3 Preconditions:
  • 6.1.54.1.0-4 LvDCDC current operating state=Buck RUNLEVEL >= 30.
  • 6.1.54.1.0-5 Trigger:
  • 6.1.54.1.0-6 Measured_U_HV <= g_dcdcinput_min_permitted_hv_voltage
  • 6.1.54.1.0-7 Input data:
  • 6.1.54.1.0-8 Target_Operating_State = Buck, Target_U_LV = 14V, Measured_U_LV = 10,5V to 13,9V. Measured_U_HV = 310V to 290V Min_Permitted_HV_Voltage = p_DCDC_MIN_U_HV = 301V, p_DCDC_DERAT_MIN_U_HV = 150V, Max_Permitted_HV_Power = 3100 W, p_DCDC_CONTINUOUS_I_LV = 221 A, p_DCDC_CONTINUOUS_P_LV = 3100 W, Coolant temperature = 35Â°C
  • 6.1.54.1.0-9 Description of behaviour:
  • 6.1.54.1.0-10 In case that the HVDC voltage is equal or lower than the threshold g_dcdcinput_min_permitted_hv_voltage given by HVPM or modificable via INCA, the LvSupplySystem stops the conversion. If the HVDC voltage is above again, then the LvDCDC shall restart. The Deratingstatus_DCDC-Wandler_Ist shall be modified accordingly with the information about the status "Spannung_DCDC-Wandler_Hochvoltteil_minimal_ Grenze" / "voltage_DCDC-converter_Highvoltagepart_minimal_limit".
