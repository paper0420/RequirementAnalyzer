#SYQT-64-0001-0004#

[Thermal_Management],[AKKS],[Basic_Operation]
Test Objective: AKKS2 Before and after learning the lower limit position Check the position move operation.

Requirements

  • 25171 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The stepper motor position (target position) shall be calculated.
  • 25181 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The stepper motor mode (normal, service, stop) shall be calculated.
  • 25186 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The speed (TURN_V_AKKS_LIN) shall be set in a configurable manner depending on the requirement (stop test OPEN, stop test CLOSED, opening, closing).
  • 25187 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The torque delimiter (SMOT_TORQ_AKKS_LIN) shall be set in a configurable manner depending on the requirement (stop test OPEN, stop test CLOSED, opening, closing).
  • 25242 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If there is a movement request and the lower stop has not been learned yet, this shall be learned and then the target position shall be approached.
  • 25243 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If there is a movement request and the lower stop has already been learned (stop test CLOSED was successful), the target position shall be approached directly.
  • 25246 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    A function for checking the upper limit stop shall be provided, which is activated (stop test OPEN) by a selectable trigger (error, vehicle status, vehicle speed, tester job, status stop test CLOSED). (see Picture 1)
  • 25248 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    An applicable vehicle speed value shall be realized as trigger for stop test OPEN.
  • 25250 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    After triggering of stop test OPEN, the target position is set to an applicable high value.
  • 25251 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    During the stop test OPEN the travel path of the flap shall be checked.
  • 25252 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If a blockade is detected within the traversing range the stop test OPEN is finished successfully.
  • 25266 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    It shall be possible to set the blocking detection ON or OFF (SMOT_BLK_AKKS_LIN) per application.
