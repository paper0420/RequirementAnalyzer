  • 7.2.2.1 SYR_NF_PMG_010: supply voltage ranges
  • 7.2.2.1.1 Description
  • 7.2.2.1.1.0-1 Brief description: depending of voltage supply level (Ubat/ KL30) the CCU has to operate like this:
  • 7.2.2.1.1.0-2 Preconditions: CCU is connected to voltage supply
  • 7.2.2.1.1.0-3 Trigger: change of voltage supply level
  • 7.2.2.1.1.0-4 Input data: voltage supply
  • 7.2.2.1.1.0-5 Description of behaviour: FALLING VOLTAGE: falling below 4.5V more than 2ms: CCU COMPLETE STOP falling below 6V more than 500ms: CCU COMPLETE STOP RAISING VOLTAGE raising to >= 6V : full operation (CAN/ DC-DC ) at once rasing to>16V more than 1ms : CCU COMPLETE STOP (DC-DC off!) In any case: if voltage revovered to 6V <= Voltage <=16V, the CCU has to enter "normal operation" within 1ms (one millisecond) if CAN-Wakeupline is active (HIGH) while voltages raises >=6V, the CCU has to operate CAN-communication within 150ms after voltage became >=6V.
  • 7.2.2.1.1.0-6 Timing Requirements: see behaviour description
  • 7.2.2.1.1.0-7 Output data: none
  • 7.2.2.1.1.0-8 Postconditions: none
  • 7.2.2.1.1.0-9 Descriptions of exceptions: none
  • 7.2.2.1.1.0-10 Dependencies and interactions: none
