  • 4.2.1.5.1.0-2 Epic# - CCU_820 Evaluation of the signal quality for pedal sensor position based on Hardware diagnostic signal measured from M_SREG_DIAG_ADC.
  • 4.2.1.5.1.0-3 Preconditions:
  • 4.2.1.5.1.0-4 Ignition key (KL15) is ON

  • 4.2.1.5.1.0-9 Description of behaviour:
  • 4.2.1.5.1.0-10 1. The CCU shall set the qualifier for sensor 1 position signal as a function of the HW diagnostic signal (M_SREG_DIAG_ADC signal): If M_SREG_DIAG_ADC signal signalizes an error Then Set BMW_u_AccrSnsr1_qi equal to 15: signal value invalid Else Set BMW_u_AccrSnsr1_qi equal to 0: signal OK Implementation Hint: For B phase the qualifier for the sensor 1 position signal could have the values “0: signal OK” or “15: signal value invalid”. 2. The CCU shall set the qualifier for sensor 2 position signal as a function of the HW diagnostic signal (M_SREG_DIAG_ADC signal): If M_SREG_DIAG_ADC signal signalizes an error Then Set BMW_u_AccrSnsr2_qi equal to 15: signal value invalid Else Set BMW_u_AccrSnsr2_qi equal to 0: signal OK Implementation Hint: For B phase the qualifier for the sensor 2 position signal could have the values “0: signal OK” or “15: signal value invalid”.
