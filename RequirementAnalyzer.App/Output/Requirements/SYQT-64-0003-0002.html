#SYQT-64-0003-0002#

[Thermal_Management],[AKKS],[Basic_Operation]
Test Objective: Check AKKS2 operation when CCU starts up.

Requirements

  • 25171 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The stepper motor position (target position) shall be calculated.
  • 25181 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The stepper motor mode (normal, service, stop) shall be calculated.
  • 25186 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The speed (TURN_V_AKKS_LIN) shall be set in a configurable manner depending on the requirement (stop test OPEN, stop test CLOSED, opening, closing).
  • 25187 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The torque delimiter (SMOT_TORQ_AKKS_LIN) shall be set in a configurable manner depending on the requirement (stop test OPEN, stop test CLOSED, opening, closing).
  • 25236 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    An applicable time value after Ignition ON shall be realized as trigger for stop test CLOSED.
  • 25237 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    After triggering of stop test CLOSED, the target position is set to an applicable low value.
  • 25238 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If a blockade is detected within the traversing range the stop test CLOSED has finished successfully.
