  • 3.1.21 SYR_HMG_77: [AKKS] Separate Blockade-DTC and FrozenFlap-DTC
  • 3.1.21.1 Description
  • 3.1.21.1.0-1 Brief description: Depending on the ambient temperature either a Blockade-DTC or a FrozenFlap-DTC shall be set when detecting a mechanical blockade of AKKS.
  • 3.1.21.1.0-2 Preconditions: Kl15 ON.
  • 3.1.21.1.0-3 Trigger: AKKS blockade-event counter exceeds configured threshold.
  • 3.1.21.1.0-4 Input data: - Blockage event counter - Ambient temperature
  • 3.1.21.1.0-5 Description of behaviour: - Blockade DTCs (0x224040, 0x224046) shall be set if (ambient temperature > Thresh_Fail_AKKSx_FrozenFlap_Temp) in primary error memory - Frozen-Flap DTCs (0x22404C, 0x224056) shall be set if (ambient temperatures <= Thresh_Fail_AKKSx_FrozenFlap_Temp) in secondary error memory
  • 3.1.21.1.0-6 Timing Requirements: NA
  • 3.1.21.1.0-7 Output data: Error_Pack_AKKS_Blockade
  • 3.1.21.1.0-8 Postconditions: NA
  • 3.1.21.1.0-9 Descriptions of exceptions: AKKS1 is not mounted for SP21 car lines.
  • 3.1.21.1.0-10 Dependencies and interactions: BSW (DEM) Related Epics: CCU-28949, CCU-29015, CCU-29048, (CCU-3740)
