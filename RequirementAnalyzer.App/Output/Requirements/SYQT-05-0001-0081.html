#SYQT-05-0001-0081#

[Charging],[AC_Charging],[AC_Charging_Control]
Test Objective: Enable Dynamic Change between 1 and 3 phase configuration (CR-1156) (CCU-32487)

Requirements

  • 9840 -
    [Not Changed][partly accepted][SYQT (ENG.10)][Requirement]
    The component "CCU" must integrate the technical module "switching matrix" for switching from one to three phase charging in the charger.
  • 9841 -
    [Not Changed][partly accepted][SYQT (ENG.10)][Requirement]
    The component "CCU" must recognize, whether a one phase or a three phase charging station is connected to the vehicle to be able to use the maximum available charge power supported by the vehicle.
  • 17458 -
    [Not Changed][accepted][SYIT (ENG.9)][Requirement]
    The "Ladekoordinator" function shall command the configuration of the switch matrix (single-phase or three-phase) of the "Ansteuerung_Ladegeraet AC_DC_Hochvolt" function via the "Soll_Modus_Schaltmatrix" signal.
  • 17459 -
    [Not Changed][accepted][SYIT (ENG.9)][Requirement]
    Depending on the commanded "Soll_Modus_Schaltmatrix", the "Ansteuerung_Ladegeraet AC_DC_Hochvolt" function shall configure the switch matrix to single-phase or three-phase charging.
  • 17460 -
    [Not Changed][accepted][SYIT (ENG.9)][Requirement]
    If single-phase charging has been commanded, the "Ansteuerung_Ladegeraet AC_DC_Hochvolt" function shall configure its switch matrix such that charging module 2 and if a second switch matrix is installed, charging module 3 are coupledto phase 1.
  • 17461 -
    [Not Changed][accepted][SYIT (ENG.9)][Requirement]
    If three-phase charging has been commanded, the "Ansteuerung_Ladegeraet AC_DC_Hochvolt" function shall configure its switch matrix such that charging module 1 is coupled to phase 1, charging module 2 to phase 2 and charging module 3 to phase 3.
  • 17462 -
    [Not Changed][accepted][SYIT (ENG.9)][Requirement]
    Once a new configuration has been specified via "Soll_Modus_Schaltmatrix", the "Ansteuerung_Ladegeraet AC_DC_Hochvolt" function shall adopt this target condition within 100Â ms.
