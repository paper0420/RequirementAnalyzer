#SYQT-58-0001-0008#

[Device_Control],[3way_Valve],[3way_Valve_Control]
Test Objective: Valve opening closing indication value can be transmitted with LIN

Requirements

  • 26982 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The function must transmit the opening angle control signal as a LIN message to the valve.
  • 26983 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The value of the opening must be able to receive between 0 and 1000.
  • 26985 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    All values outside the values 0 to 1000 shall be evaluated invalid.
  • 26986 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The response values defined in the message catalog (Jira BN-31615) must be able to be output as control unit variable.
  • 26988 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The input variable for the function is the control request from the TMC for the valve.
  • 26996 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The function must be executed in all control unit states.
