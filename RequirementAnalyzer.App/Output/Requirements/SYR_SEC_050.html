  • 7.7.1.5.1.0-4 Input data: content of flash memory
  • 7.7.1.5.1.0-5 Description of behaviour: described in https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.03_SystemArchitecturalDesign/ENG.03_SYAD_Design/SECURITY/CCU_architecture/master.pdf
  • 7.7.1.5.1.0-6 Timing Requirements: none / any IDLE time will be consumed
  • 7.7.1.5.1.0-7 Output data: result of integrity check : TRUE/FALSE
  • 7.7.1.5.1.0-8 Postconditions: none (will never terminate)
