SYR_CIN_340


  • 8.1.38 SYR_CIN_340: Add coding parameter for max DC Fahrzeugleistung
  • 8.1.38.1 Description
  • 8.1.38.1.0-1 Brief description: Coding Parameter for Max DC Fahrzeugleistung needs to be updated in Coding Module. The Max DC Fahrzeugleistung is related to the max DC vehicle charging power. This system requirement is valid for I20/G26.
  • 8.1.38.1.0-2 Preconditions: CCU in flash mode
  • 8.1.38.1.0-3 Trigger: coding CCU
  • 8.1.38.1.0-4 Input data: CAF file, FA file
  • 8.1.38.1.0-5 Description of behaviour: The Max DC Fahrzeugleistung is depending on the SOPs, country codes and HV storage. For the Coding parameter to be introduced in coding table, the client/server interface of the BAC coding module shall be converted into a S/R interface towards BMW SWC. Acceptance : The coding parameter shall be accessible to BMW SWC
  • 8.1.38.1.0-6 Timing Requirements: NA
  • 8.1.38.1.0-7 Output data: BMW coding parameter signal is set as per FA file
  • 8.1.38.1.0-8 Postconditions: The coding parameter shall be accessible to BMW SWC
  • 8.1.38.1.0-9 Descriptions of exceptions: NA
  • 8.1.38.1.0-10 Dependencies and interactions: NA
