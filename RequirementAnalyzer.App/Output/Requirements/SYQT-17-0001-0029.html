#SYQT-17-0001-0029#

[Charging],[Component_Protection],[Derating]
Test Objective: Cofirm that HVDC output power is not derated when HVDC output starts from the high temperature state (where the derating power is higher than the actual output power). (Variant:11kW with part-load charging)

Requirements

  • 18386 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If the temperature up to which the charging module shall not derate is exceeded, the derating power shall correspond to the maximum output value that the hardware can still provide.
