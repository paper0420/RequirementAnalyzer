  • 1.1.28.4.1.0-2 Detected and send current state of connector proximity detection line(A_T_Proxi) to BMW SWC (LIM-C) via "Status_Connector_Proximity_Detection" interface signal (0 = inactive, 1 = active, 2 = error). The Proximity line on CCU interface will be shared between CHADEMO Proximity and GBTDC Proximity.
