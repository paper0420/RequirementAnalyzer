  • 7.9.1.5.1.0-2 This UC gives brief description for the Class 1 signal quality signals. For Class 1 signal, Internal qualifier is determined by Bus communication monitoring and invalidity identification (COM error) when Signal Tag is not available. Internal Qualifier represents Infrastructure monitoring for signals. The data element is directly generated from the payload of the Board Network signal. The Internal qualifier is generated by COM Adapter by monitoring the signal for COM error (Timeout, CRC, Alive) and Debouncing status. Internal Qualifier consists of states Init, Not_Available, Error, Ok, Inactive, Debouncing. The Internal qualifier is transmitted atomically with in the software. The value of the signal itself represents the status of the signal. Com Adapter debounces each signal and after debouncing time is reached and error is still present COM Adapter records the DTC code through DEM. The COM Adapter generates Internal Qualifier from Signal Tags. For Group of signal having more than 1 Signal Tag the highest value is considered to generate Internal Qualifier. For example for 8 bit value 0 to 252 are taken as OK and 253 to 255 are taken as invalid. The internal qualifier is updated as 0 for OK and 15 for invalid values. Refer BMW03_SysReqSpec_COMAdapter.doc for more details on COMAdapter. https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.02_SystemRequirementsAnalysis/ENG.02_SystemRequirementSpecification/AdditionalDocuments/ENG2_Overview_Documents/BMW03_SysReqSpec_COMAdapter.docx
