#SYQT-06-0001-0015#

[Charging],[AC_Charging],[AC_Measurement_Accuracy]
Test Objective: LT4 voltage accuracy check (0V-540V) (Add negative test)

Requirements

  • 23640 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The CCU during the AC charging should ensure that the DC voltage is in the allowed ranges, that is below 540V. For this safety function: - The supplier should measure the DC voltage on LT4 within the specified range and accuracy. - Supplier will evaluate LT4 voltage, if it is higher than the allowed range then shall stop the AC charging.
  • 23642 -
    [Not Changed][accepted][SYQT (ENG.10)SYIT (ENG.9)Review][Requirement]
    The mean voltage value (over last 10ms) present on E_U_LT4 shall be sent over signal "LT4_U_Sig" with following accuracy LT4_U_Sig < 10V: +/- 1V 10V <= LT4_U_Sig < 1000V: +/- 5% Safe State : Respective Qualifier set to "signal invalid", LT4_U_Sig_qi equal to "Invalid" FTT: 160ms
  • 6375 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    For the test, the test parameters shall be selected according to the table “Test parameters of the upper restricted operational capability range” ENBN_PR_2640.
  • 6416 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Report of the Test shall be provided for all operating voltage ranges in accordance with the table “High voltage ranges” (ENBN_PR_2255).
  • 11154 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    The "Messwertermittlung_Ladegerät_AC/DC" function shall determine the DC-HV voltage output by the charging electronics.
  • 11464 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) connect the development object with resistive load. b) activate and operate the development object at the constant-voltage output mode with the output voltage being a set value Uz0 in the range of output voltage of the development object.
  • 11465 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) adjust the input voltage of the development object such that it varies within ± 15% of the rated value. b) adjust the output current such that it varies within the range between no load to rated current. b) measure actual output voltage Uz.
  • 17648 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) output voltage tolerance must be not more than ± 1% calculated per formula: Of which, ξU -- Output voltage tolerance; Uz -- Actual output voltage (V); Uz0 -- Set output voltage (V).
