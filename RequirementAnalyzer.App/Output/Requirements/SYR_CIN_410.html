SYR_CIN_410


  • 8.1.45 SYR_CIN_410: Add coding parameter for Control Horn and Light depending on country type
  • 8.1.45.1 Description
  • 8.1.45.1.0-1 Brief description: Coding Parameters for providing the horn and the lights dependancy on country type. This system requirement is valid for I20/G26.
  • 8.1.45.1.0-2 Preconditions: CCU in flash mode
  • 8.1.45.1.0-3 Trigger: coding CCU
  • 8.1.45.1.0-4 Input data: CAF file, FA file
  • 8.1.45.1.0-5 Description of behaviour: HV_Warn_Hupen_Blinken coding parameter relates to a HV warning concept and is about the behavior of horn and lights (on, off) depending on country type. For the Coding parameter to be introduced in coding table, the client/server interface of the BAC coding module shall be converted into a S/R interface towards BMW SWC. Acceptance : The coding parameter shall be accessible to BMW SWC
  • 8.1.45.1.0-6 Timing Requirements: NA
  • 8.1.45.1.0-7 Output data: BMW coding parameter signal is set as per FA file
  • 8.1.45.1.0-8 Postconditions: The coding parameter shall be accessible to BMW SWC
  • 8.1.45.1.0-9 Descriptions of exceptions: NA
  • 8.1.45.1.0-10 Dependencies and interactions: NA
