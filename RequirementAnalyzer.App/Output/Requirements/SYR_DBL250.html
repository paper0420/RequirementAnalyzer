  • 7.6.1.25 SYR_DBL250: pointer based calibration - calculate checksum
  • 7.6.1.25.1 Description
  • 7.6.1.25.1.0-1 Brief description: Calibration tool requests checksum calculation over calibration data using refPageAddress range. For definition and terms see also: https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.03_SystemArchitecturalDesign/ENG.03_SystemArchitecturalDesignSpecification/AdditionalDocuments/Calibration/bmw_ccu_calibration_architecture.pptx
  • 7.6.1.25.1.0-2 Preconditions: XCP session established
  • 7.6.1.25.1.0-3 Trigger: XCP calculate checksum command containing refPageAddressStart and refPageAddressEnd
  • 7.6.1.25.1.0-4 Input data: refPageAddressStart and refPageAddressEnd
  • 7.6.1.25.1.0-5 Description of behaviour: (1) XCP Slave iterates through OriginPointerTable using the following algorithm (pseudo code): refPageAddressTemp := refPageAddressStart checksum = init_checksum() while (refPageAddressTemp < refPageAddressEnd) do size = getsize(originPointerTable, refPageAddressTemp) slotindex = getSlot(originPointerTable, refPageAddressTemp) workaddress = getAddress(onlinePointerTable, slotIndex) checksum = calc_checksum(checksum, workaddress, size) refPageAddressTemp = refPageAddressTemp + size end (2) Return calculated checksum
  • 7.6.1.25.1.0-6 Timing Requirements: command execution shall occur at low priority.
  • 7.6.1.25.1.0-7 Output data: calculated checksum
  • 7.6.1.25.1.0-8 Postconditions: None
  • 7.6.1.25.1.0-9 Descriptions of exceptions: Use case always succeeds
