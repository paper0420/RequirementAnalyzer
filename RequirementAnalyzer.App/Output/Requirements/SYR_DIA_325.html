  • 7.5.1.33 SYR_DIA_325: collect Reset DTC environmental data
  • 7.5.1.33.1 Description
  • 7.5.1.33.1.0-1 Brief description: Environmental data for the DTC (ID: 0xBA0023 = Sammelfehler Software) has to be collected during normal CPU operation. The collected data has to be provided in a RAM area that is retained from operation cycle i to operation cycle i+1 with a CPU reset between cycle i and cylce i + 1. Collection occurs either periodically or event driven (preferred) based on collected data. Baisc guideline: CPU load shall be as small as possible. Note: New environmental variables was introduced in I20 variant. The list of all environmental variables can be found in I20 Zedis (version 25).
