SYR_CIN_310


  • 8.1.35 SYR_CIN_310: Provide E/E support for Autonomous Driving by coding parameter
  • 8.1.35.1 Description
  • 8.1.35.1.0-1 Brief description: Autonomous Driving support is not needed in in All Vehicle variants. Based on CODING Parameter for Autonomous Driving the PDUs which are to be received from "hPAD_Safety" control unit has to be Disabled or Enabled in CCU.
  • 8.1.35.1.0-2 Preconditions: NA
  • 8.1.35.1.0-3 Trigger: NA
  • 8.1.35.1.0-4 Input data: CAF , BMW NK
  • 8.1.35.1.0-5 Description of behaviour: Based in Coding parameter hPadHardware having possible value "verbaut " or "nicht verbaut" the "hPAD_Safety" control unit will be present in the Vehicle. So Based on CODING Parameter monitoring of "hPAD_Safety" control unit related PDUs in CCU has to be enabled or Disabled. Acceptance : - Coding accesible to SWC - No Timeout, CRC , Alive Error monitoring for PDUs from "hPAD_Safety" control unit when hPadHardware is not Available ("nicht verbaut") - Timeout , CRC, Alive Monitoring and Siganal Values are provided to SWC when hPadHardware is Available ("verbaut")
  • 8.1.35.1.0-6 Timing Requirements: NA
  • 8.1.35.1.0-7 Output data: NA
  • 8.1.35.1.0-8 Postconditions: Acceptance : - Coding accesible to SWC - No Timeout, CRC , Alive Error monitoring for PDUs from "hPAD_Safety" control unit when hPadHardware is not Available ("nicht verbaut") - Timeout , CRC, Alive Monitoring and Siganal Values are provided to SWC when hPadHardware is Available ("verbaut")
  • 8.1.35.1.0-9 Descriptions of exceptions: NA
