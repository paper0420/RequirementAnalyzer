  • 6.1.27 SYR_LvDCDC_F_250: Continuous power delivery of 3550W at an actual LV range of 14V to 15,5V and Tcool = 35°C
  • 6.1.27.1 Description
  • 6.1.27.1.0-1 Brief description:
  • 6.1.27.1.0-2 The LVSupplySystem transfers energy when the target voltage is higher than the measured voltage. Between 14V and 15,5V the limiting factor shall be the power. The consumer current and the actual voltage detemine the power.
  • 6.1.27.1.0-3 Preconditions:
  • 6.1.27.1.0-4 Current_Operating_State=Buck or Standby, RUNLEVEL >= 30, No error
  • 6.1.27.1.0-5 Trigger:
  • 6.1.27.1.0-6 Target_Operating_State = Buck, Target_U_LV = 15V, The electronic DC load power is adjusted to 3550 W The electronic DC load voltage is adjusted to 15V. The regulation mode of the electronic DC load shall be constant power.
  • 6.1.27.1.0-7 Input data:
  • 6.1.27.1.0-8 Target_Operating_State = Buck, Target_U_LV = 15V, Measured_U_LV = 14V Measured_U_HV = 380V Min_Permitted_HV_Voltage = 300V, Max_Permitted_HV_Power (Leistung_DCDC-Wandler_Hochvoltteil_maximal_Grenze) = 3905 W, p_DCDC_CONTINUOUS_I_LV = 254A, p_DCDC_CONTINUOUS_P_LV = Coolant temperature = 35°C
  • 6.1.27.1.0-9 Description of behaviour:
  • 6.1.27.1.0-10 Same kind of test as described in the test "Transfering power between 14V and 15,5V". At this voltage range the transfer of power is limited by the permitted input power from the high voltage battery. The values cover the (KLH 14133] at 35°C.

  • 6.1.27.1.0-12 to be defined

  • 6.1.27.1.0-14 Current_Operating_State = Buck, Measured_LVcurrent = 254 A, Measured_Hvcurrent >= 9A to 10,3A Measured_U_HV = 380V, Measured_LVvoltage = 14 V, Max_Available_Lvcurrent = 0A, Current_Workload = 0 percent Other values to be reported: Yes Measured_Critical_Temperature, Measured_Coolant_Temperature, Current_Derating_Status
