  • 7.2.1.15 SYR_PMG_160: init hardware
  • 7.2.1.15.1 Description
  • 7.2.1.15.1.0-1 Brief description:
  • 7.2.1.15.1.0-2 This UC contains the configuration of hardware and software. At first the HW will be initialized by hw-processes, within this UC the software will take over and execute the higher-level hw-configuration (e.g. booting of PLC-chip, configuration via I2C/SPI) If this UC has passed, the whole HW shall be able to be used by software on application level. This UC will check if dedicated HW-component behaves correctly if possible. In case of detected failure, this will be considered in “RUN-Level” selection, to prevent usage of failed component.
  • 7.2.1.15.1.0-3 Preconditions: CCU was not connected to power
  • 7.2.1.15.1.0-4 Trigger: connect CCU to power
  • 7.2.1.15.1.0-5 Input data: none
  • 7.2.1.15.1.0-6 Description of behaviour: CCU's hardware (ALL!) will be initialised first time. This affects HW and SW components. After initilisation, the CCU is NOT executing any active tasks.
  • 7.2.1.15.1.0-7 Timing Requirements: none
  • 7.2.1.15.1.0-8 Output data: none
  • 7.2.1.15.1.0-9 Postconditions: none
  • 7.2.1.15.1.0-10 Descriptions of exceptions: none
  • 7.2.1.15.1.0-11 Dependencies and interactions: none

  • 7.2.1.16 SYR_PMG_170: measure time - obsolete
  • 7.2.1.16.1 Description
  • 7.2.1.16.1.0-1 Brief description:
  • 7.2.1.16.1.0-2 This UC will measure absolute and relative time. It’s important, that this UC is not affected, if CCU is in sleep-mode.
  • 7.2.1.16.1.0-3 Preconditions: CCU has need to sleep for certain time (e.g. Charging-Pause / cyclic reset of CPU (->FUSA))
  • 7.2.1.16.1.0-4 Trigger: application has calculated time to sleep and requests sleep of CCU for that time
  • 7.2.1.16.1.0-5 Input data: time to sleep
  • 7.2.1.16.1.0-6 Description of behaviour: time will be measured and wake up event will be generated if elepsed
  • 7.2.1.16.1.0-7 Timing Requirements: none ;-)
  • 7.2.1.16.1.0-8 Output data: wake up trigger (SSWU)
  • 7.2.1.16.1.0-9 Postconditions: Chorus CPU is running
  • 7.2.1.16.1.0-10 Descriptions of exceptions: time measure will be stopped in case of HW-reset of Chorus
