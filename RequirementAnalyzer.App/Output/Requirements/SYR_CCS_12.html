SYR_CCS_12


  • 1.1.31.2 SYR_CCS_12: provide status and error notification via software interface for respective protocol sequence between EVSE and CCU
  • 1.1.31.2.1 Description
  • 1.1.31.2.1.0-1 Brief description: Panasonic software should provide the status and error notification via software signals. Different protocol sequence would be supported by CCU. DIN DC protocol (G08) - DIN DC Protocol signal will provide the status an/Ierror of sequence between EVSE and CCU. ISO AC protocol EIM (payment mode external) (G08/I20) - ISO AC Protocol signal will provide the status and error of sequence between EVSE and CCU. ISO AC protocol PnC (payment mode contract) (tbd) - ISO AC Protocol signal will provide the status and error of sequence between EVSE and CCU. ISO DC protocol EIM (payment mode external) (tbd) - ISO DC Protocol signal will provide the status and error of sequence between EVSE and CCU. ISO DC protocol PnC (payment mode contract) (tbd) - ISO DC Protocol signal will provide the status and error of sequence between EVSE and CCU. New Priority(BMW03-25763): Pro 1: DIN Pro 2: ISO ED1 Pro 3: ISO ED2 Pro 4: reserved. Note: The signal discription is tbd, it should be received from BMW. refer SYR_PLC_03 for figure.
  • 1.1.31.2.1.0-2 Preconditions: PLC Dlink = link, all value received is valid. PLC DLink = no link, all the interface values should be considered as invalid. Possible we should receive init values.
  • 1.1.31.2.1.0-3 Trigger: BMW Software (LIM-C) will provide trigger over signals (sigHLCStart = Active)
  • 1.1.31.2.1.0-4 Input data: information via Vector callbacks/callouts
  • 1.1.31.2.1.0-5 Description of behaviour: at present vector deliver (or any other vendor) SCC package receive information via callbacks. BMW LIM-C SWC will receive information via RTE signals. The Panasonic software should collect all the information from vector callbacks and provide in BMW Signal format as defined in signal description file (tbd). This function should be as stateless as possible.

  • 1.1.31.2.1.0-7 Output data: RTE signals to BMW LIM-C

  • 1.1.31.2.1.0-9 Descriptions of exceptions: In G08, only DIN DC is implemented. In I20, ISO AC is implemented
