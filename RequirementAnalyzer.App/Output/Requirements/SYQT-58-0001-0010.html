#SYQT-58-0001-0010#

[Device_Control],[3way_Valve],[3way_Valve_Control]
Test Objective: The ability to control the valve according to whether the assembly mode is enabled or disabled

Requirements

  • 26996 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The function must be executed in all control unit states.
  • 26998 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The function must include the status before activating the valve.
  • 27000 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The valve must not be activated when the assembly mode is activated.
  • 27001 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The valve must be able to be activated in the assembly mode.
