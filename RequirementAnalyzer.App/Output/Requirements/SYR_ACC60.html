  • 2.1.6 SYR_ACC60: Function: "Messwertermittlung"-measure input HV AC current
  • 2.1.6.1 Description
  • 2.1.6.1.0-1 Brief description:
  • 2.1.6.1.0-2 determine the actual root‐mean‐square value of AC current between the connected conductor and the neutral conductor of the charger.
  • 2.1.6.1.0-3 Preconditions:
  • 2.1.6.1.0-4 none (always)
  • 2.1.6.1.0-5 Trigger:
  • 2.1.6.1.0-6 start up (always active)
  • 2.1.6.1.0-7 Input data:
  • 2.1.6.1.0-8 E_I_HVAC_L1Phy E_I_HVAC_L2Phy E_I_HVAC_L3Phy
  • 2.1.6.1.0-9 Description of behaviour:
  • 2.1.6.1.0-10 measure mean value of one monitoring cycle for the different phases over one monitoring cycle. Distribute over the respective signals. The required accuracy should be respected. Safety requirements: To be implemented as ASIL B Mean error probability: 10 FIT outage metrics: SPFM: > 90% ; LFM: > 60% Implementation hint. Signal names for values distribution: * send E_I_HVAC_L1Phy to signal "HVAC_I_L1Sig" * send E_I_HVAC_L2Phy to signal "HVAC_I_L2Sig" * send E_I_HVAC_L3Phy to signal "HVAC_I_L3Sig" accuracy: 0A < E_I_HVAC_L1Phy<100A: +/‐ 3A Conversion formula : IrmsNotAdjust = ( ADC_HVAC_Lx_Curr * 4.096V / 2^12 ) * ( Pi/ 2*SQRT(2) ) * ( 1 / 22.4166666666667 ) * ( 1 / 0.003 ) HVAC_Lxx_Curr = ( 2.5 * (Vac -70)) / (1000 * 1000) * (IrmsNotAdjust - 18) ^2 + 0.95 * IrmsNotAdjust Filter, Debounce : moving average 8 times Safe State : Respective Qualifier set to not valid.
