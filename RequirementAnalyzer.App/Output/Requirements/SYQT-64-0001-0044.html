#SYQT-64-0001-0044#

[Thermal_Management],[AKKS],[Error_Handling]
Test Objective: Debounce process failed due to block detection in Stp Test Open and abnormally end.

Requirements

  • 25262 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    A function shall be provided, which detects a blocked actuator (blocking during the valid travel path) and cyclically starts break-off tests with strike out (maximum number applicable).
  • 25263 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    Strike out means the repeated attempt to reach the detected blockade position. The position shall be approached from a suitable distance to shake the blockade free.
  • 25265 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the maximum number of break-off tests is reached, an error memory shall (DFC) be set and the actuator shall move back to an applicable position. (see table 1)
  • 25270 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    As reaction for one or more errors the actuator must be able to go to an applicable position.
