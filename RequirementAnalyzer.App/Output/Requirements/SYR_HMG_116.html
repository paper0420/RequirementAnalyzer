  • 3.1.53 SYR_HMG_116: [S5] Startup-WaitTime for WJ-S5 Plausibility check
  • 3.1.53.1 Description
  • 3.1.53.1.0-1 Brief description: The plausibility check S5-temperature sensor vs. WJ-temperature sensor shall start with a configurable time-delay after enabling the check. This time-delay shall also be applied if the check is enabled at CCU startup. Typical value for time-delay: 5min.
  • 3.1.53.1.0-2 Preconditions: Kl15 ON.
  • 3.1.53.1.0-3 Trigger: ECU Startup.
  • 3.1.53.1.0-4 Input data: - Control signal 'BMW_b_DiCoRadrOutPlauDiag' received from BMW ASW (corresponds to signal 'Enable_S5WJ_PlausiCheck'; (0 = Enabled; 1=Disabled) - Calibration label: WaitTime_For_ENA_S5WJ_PLAUSI
  • 3.1.53.1.0-5 Description of behaviour: - The plausibility check S5- vs. WJ-temperature sensor shall be inactive when Enable_S5WJ_PlausiCheck = Disabled(1). - The plausibility check shall start with a configurable time-delay ‘WaitTime_For_ENA_S5WJ_PLAUSI’ after a transition of ‘Enable_S5WJ_PlausiCheck’ from Disabled(1) to Enabled(0). After this time-delay, the check shall be active. The time-delay for activation of the check shall also be applied if the check is enabled at CCU startup.
  • 3.1.53.1.0-6 Timing Requirements: See description of behaviour.
  • 3.1.53.1.0-7 Output data: Plausibility check result.
