  • 7.2.1.17 SYR_PMG_190: shutdown device
  • 7.2.1.17.1 Description
  • 7.2.1.17.1.0-1 Brief description:
  • 7.2.1.17.1.0-2 This UC gathers all actions, needed to execute a safe shutdown of CCU. It reflects the “shutdown path” of SYR_PMG140. While shut down it’s checked if all components are ready to shut down. In case of delay by some component, this UC will monitor this component and force stop of this component if necessary. This UC will generate log entries in persistent log-buffer for offline analysis of problems.
  • 7.2.1.17.1.0-3 Preconditions: power management decides to shutdown CCU
  • 7.2.1.17.1.0-4 Trigger: request to enter RUNLEVEL0
  • 7.2.1.17.1.0-5 Input data: none
  • 7.2.1.17.1.0-6 Description of behaviour: CCU will enter minimal power consumtion
