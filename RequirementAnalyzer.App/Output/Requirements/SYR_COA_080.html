  • 7.9.1.13 SYR_COA_080: COM Adapter End to End protection mechanism
  • 7.9.1.13.1 Description
  • 7.9.1.13.1.0-1 Brief description: This UC gives brief description of the End to End protection using Time out monitoring for sensor signals and onboard electrical signals generate corresponding Qualifiers by COM Adapter. During the generation of qualifiers COM Adapter monitors the Time out, CRC, Alive status with debounce counters for predefined duration of time and update the qualifier status.
  • 7.9.1.13.1.0-2 Preconditions: RUNLEVEL>=30
  • 7.9.1.13.1.0-3 Trigger: COM Adapter receives Signal or Group signals from COM
  • 7.9.1.13.1.0-4 Input data: c
  • 7.9.1.13.1.0-5 Description of behaviour: COM adapter shall provide End to End protection by monitoring received COM signals and assigning Signal Qualifiers depending on the monitoring status.
  • 7.9.1.13.1.0-6 Timing Requirements: (Message cycle time + tolerance) ms
  • 7.9.1.13.1.0-7 Output data: When input data is valid COM Adapter transfers COM signals with Qualifier information When input data is invalid COM Adapter transfers invalidity value.
