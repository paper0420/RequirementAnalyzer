  • 7.9.1.10.1.0-2 This UC gives brief description for the Class 3 signal quality signals with no Signal Qualifier, no Internal Qualifier and no signal Tag. These signal are communicated as discrete signals in network and application. ComAdapter does not add any additional information for valid signals. If the signal received from COM is no valid then invaidity value is transfered by COM adapter to the application.
