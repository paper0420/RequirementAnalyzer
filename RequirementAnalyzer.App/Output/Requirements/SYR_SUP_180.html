  • 7.4.2.19 SYR_SUP_180: Prevent operation
  • 7.4.2.19.1 Description
  • 7.4.2.19.1.0-1 Brief description:
  • 7.4.2.19.1.0-2 This USE-case will take care to prevent any active operation of CCU’s nominal tasks while flashing is ongoing. At startup of flash-procedure all HW-components will be set to defined state of “passiveness” to prevent any CCU’s driven activity to any other interfaces than communication interface used for flashing. While flashing, this Use-Case will supervise to ensure no accidently operation of any non-flash-related hardware.
  • 7.4.2.19.1.0-3 Preconditions: CCU is in flash-mode
  • 7.4.2.19.1.0-4 Trigger: start of Bootloader
  • 7.4.2.19.1.0-5 Input data: none
  • 7.4.2.19.1.0-6 Description of behaviour: CCU will stop any HV/LV activities and ensure no of this operations untill the Bootloader has been stopped an main-application is unter execution again
  • 7.4.2.19.1.0-7 Timing Requirements: 0ms
  • 7.4.2.19.1.0-8 Output data:none
  • 7.4.2.19.1.0-9 Postconditions: HV/LV energy flow stopped
  • 7.4.2.19.1.0-10 Descriptions of exceptions: none
  • 7.4.2.19.1.0-11 Dependencies and interactions: none
