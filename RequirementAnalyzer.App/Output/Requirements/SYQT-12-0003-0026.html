#SYQT-12-0003-0026#

[Charging],[14V_DCDC],[DCDC_Control]
Test Objective: Confirm peak current of LVDC output is limited current within calibration value.(Limiting DCDC Output current gradient to 100A/s)

Requirements

  • 14530 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Gleichspannungswandlung" shall limit the LV current to "p_DCDC_PEAK_I_LV" if the "DC/DC" converter function is in "Buck" operating mode.
  • 14531 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Gleichspannungswandlung" shall limit the LV current to "p_DCDC_CONTINUOUS_I_LV" if the "DC/DC" converter function is in "Buck" operating mode.
