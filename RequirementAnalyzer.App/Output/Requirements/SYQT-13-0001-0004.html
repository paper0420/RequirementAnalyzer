#SYQT-13-0001-0004#

[Charging],[14V_DCDC],[DCDC_Measurement_Accuracy]
Test Objective: Confirmation of LV current measurement value (Activation from AC charge)

Requirements

  • 23932 -
    [Not Changed][accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    The mean current value (over last 100ms) present on H_I_DCDC shall be sent over signal "LV_DCDC_I_Sig" with following accuracy LV_DCDC_I_Sig < 500A: +/- 5A Safe State : Respective Qualifier set to not valid, LV_DCDC_I_Sig_qi equal to "Invalid" FTT: 100ms
  • 18119 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    In addition to the requirements described here, the "DC/DC" function (DC/DC conversion) shall comply with all the functionally relevant requirements for the DC/DC converter from section 5.4.1. in the requirements specification "Hochvoltpowermanagement Client (HVPM)" [LH 1023 1491:V01].
  • 18241 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The "DC/DC" function (DC/DC conversion) shall comply with the values described in the Table "Accuracy parameters for output variables" to ensure the accuracy of the output variables.
  • 18242 -
    [Not Changed][partly accepted][SYQT (ENG.10)][Requirement]
    "Accuracy parameters for output variables" Output variable x d Actual voltage on HV-side 1 % 0.5 V Actual current on HV-side 1 % 0.1 A Actual voltage on NV-side 1 % 0.05 V Actual current on NV-side <6 % 1 A x: Relative accuray in per cent (+/-) of the physical value d: Maximal inaccuray (+/-) independent from the measured value
