  • 7.11.1.2 SYR_SF19 PFM alive monitoring tolerance reduction
  • 7.11.1.2.1 Description
  • 7.11.1.2.1.0-1 Brief description: 1. Configure WDG “Failed alive Supervision” configuration parameter to “0” for ASIL TASK 2. Set WDG Max Margin and Min Margin configuration within "0%" for ASIL TASK(Any tolerance devaition in ASIL TASK need to be justified with Safety impact analysis and agreement with BMW and Panasonic FuSa) 3. If the above configuration causing WDG RESET, revisit the current TASK scheduling and TASK priority It is highly risky to have any tolerance in FuSa TASK as this may impact fault tolerance time- FTT, safe state and which may leads to safety goal violation.
