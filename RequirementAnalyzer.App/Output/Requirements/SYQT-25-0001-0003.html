#SYQT-25-0001-0003#

[Charging],[Component_Protection],[DC_Undervoltage]
Test Objective: Low Voltage Detection Confirmation of HVDC Output Voltage AC -> HV battery (3-phase)

Requirements

  • 6423 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Compliance with the high voltage operating condition B3/B4 for overvoltage shall be demonstrated, see table “High voltage operating conditions (ENBN_PR_2277)”.
  • 6426 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The voltage increase and fall for the test pulse overvoltage shall occur with the maximum voltage dynamics in accordance with the table “Maximum voltage dynamics”(ENBN_PR_2550)..
  • 6432 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    For the test, the test parameters shall be selected according to the table “Test parameters for overvoltage” (ENBN_PR_2730).
  • 6434 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    For the test, the test pulse shall be selected according to the figure “Test pulse for overvoltage” (ENBN_PR_2732).
  • 6449 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    During the Test the objekt of development must create an error memory entry with the content Undervoltage, if an output requirement is present
  • 18393 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The "Komponentenschutz und Derating" function shall detect the following protection conditions (detect errors and monitor limit values) in order to adjust operation of the charging electronics without damaging the hardware: - AC undervoltage - DC-HV undervoltage - AC overvoltage - DC-HV overvoltage - Maximum AC current - Maximum DC-HV current - Hardware fault (e.g. short circuit) - Excess temperature - Broken N-Line
  • 18405 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The function "Komponentenschutz und Derating" shall detect a DC-HV undervoltage error if the charger measures a DC voltage below the DC voltage operating limit for a parameterizable period.
  • 18406 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If a DC undervoltage is detected, the function "Komponentenschutz und Derating" shall the "Abschaltung Unterspannung DC-HV" bit error status word.
  • 18407 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    These diagnostics shall be performed only if the "Charge" target operating mode (2) is commanded by the "LDK" function on the respective charging module.
  • 18415 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The diagnostics for DC-HV overvoltage diagnostics shall be performed in every operating state.
