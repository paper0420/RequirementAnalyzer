  • 7.4.1.21 SYR_SWU_020: startup
  • 7.4.1.21.1 Description
  • 7.4.1.21.1.0-1 Brief description:
  • 7.4.1.21.1.0-1.0-1 The bootloader will be started: This include the initialisation of complete hardware and operating system. In this mode the CCU is ready to update the main-application via diagnosis communication The CCU is able to operate restricted power-management and communication via all external busses. The diagnosis-application is available partly: Only flash-update relevant services will be available. The bootloader will set all “charging related” components to a safe, idle mode and will NOT operate them at any time.
  • 7.4.1.21.1.0-2 Preconditions:
  • 7.4.1.21.1.0-2.0-1 The device is in one of the following system states: SOC is powered and BootManager decided to start bootloader.
  • 7.4.1.21.1.0-3 Trigger:
  • 7.4.1.21.1.0-3.0-1 Boot-manager decision
