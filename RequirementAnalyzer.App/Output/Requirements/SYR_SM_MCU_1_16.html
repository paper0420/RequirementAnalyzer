Factors notified to the FCCU can be processed by the EOUT signal, but when SWT occurs, it is necessary to handle by direct interrupt processing instead of EOUT. Additional requirement required for : 1. Boot monitoring using SWT Before going to sleep mode, SWT can be configured for boot time monitoring. If timer expired, destructive reset to be performed for BIST.
