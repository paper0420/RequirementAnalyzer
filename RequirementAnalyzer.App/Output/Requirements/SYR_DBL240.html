SYR_DBL240


  • 7.6.1.24 SYR_DBL240: pointer based calibration - write calibration data
  • 7.6.1.24.1 Description
  • 7.6.1.24.1.0-1 Brief description: Calibration tool write calibration data using refPageAddress and data. For definition and terms see also: https://svn-bmw03-pais.eu.panasonic.com/BMW03_CCU_DOC/Common/WorkSpace/ENG.03_SystemArchitecturalDesign/ENG.03_SystemArchitecturalDesignSpecification/AdditionalDocuments/Calibration/bmw_ccu_calibration_architecture.pptx
  • 7.6.1.24.1.0-2 Preconditions: XCP session established
  • 7.6.1.24.1.0-3 Trigger: XCP write command from calibration tool containing refPageAddress and data.
  • 7.6.1.24.1.0-4 Input data: refPageAddress and data
  • 7.6.1.24.1.0-5 Description of behaviour: (1) XCP slave searches for slotIndex in OriginPointerTable using refPageAddress (2) XCP slave write data using pointer in onlinePointerTable using slotIndex Data can only be written to working page. If pointer from onlinePointerTable points into non-volatile memory the calibration access is terminated and a negative response is sent to the calibration tool.
  • 7.6.1.24.1.0-6 Timing Requirements: command execution shall occur at low priority.
  • 7.6.1.24.1.0-7 Output data: success indication towards calibration tool.
  • 7.6.1.24.1.0-8 Postconditions: Data in working page is changed and available to SW.
  • 7.6.1.24.1.0-9 Descriptions of exceptions: If onlinePointerTable entry points into reference page (ROM) the write access is stopped and a negative response is returned to calibration tool.
