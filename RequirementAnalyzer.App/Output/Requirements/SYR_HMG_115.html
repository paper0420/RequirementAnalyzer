SYR_HMG_115


  • 3.1.52 SYR_HMG_115: [S5] Bypass MovingAverage Filter for eTMC during Startup
  • 3.1.52.1 Description
  • 3.1.52.1.0-1 Brief description: At startup/wakeup of the CCU the unfiltered raw-value of the S5 temperature sensor shall be provided to BMW-application software by bypassing the implemented moving-average filter for a configurable time.
  • 3.1.52.1.0-2 Preconditions: Kl15 ON.
  • 3.1.52.1.0-3 Trigger: ECU Startup.
  • 3.1.52.1.0-4 Input data: - S5 temperature sensor raw/unfiltered measurement readings - S5 temperature sensor filtered measurement readings - Calibration data: BypassActivationTime, LowerThreshold for range check, UpperThreshold for range check
  • 3.1.52.1.0-5 Description of behaviour: For a configurable time-period (BypassActivationTime) beginning at startup/wakeup of CCU: - Thmgt shall provide the raw-value of the S5 temperature-sensor to eTMC via interface BMW_t_CoRadrOut - Thmgt shall set the related qualifier BMW_t_CoRadrOut_qi (a) to (OK=0) if the raw-value is within a configurable range (between LowerThreshold and UpperThreshold) (b) to (ERROR=15) if the raw-value is not within the before-mentioned range After the configurable time-period (BypassActivationTime) has elapsed - Thmgt shall provide a filtered value of the S5 temperature-signal to eTMC via interface BMW_t_CoRadrOut (moving-average filter as implemented) - Thmgt shall set the related qualifier BMW_t_CoRadrOut_qi depending on the internal state of the moving-average filter (set qualifier as implemented)
  • 3.1.52.1.0-6 Timing Requirements: See description of behaviour.
  • 3.1.52.1.0-7 Output data: S5 temperature reading and related qualifier (BMW_t_CoRadrOut, BMW_t_CoRadrOut_qi)
  • 3.1.52.1.0-8 Postconditions: NA
  • 3.1.52.1.0-9 Descriptions of exceptions: NA
  • 3.1.52.1.0-10 Dependencies and interactions: BMW_SWC_eTMC Related Epics: CCU-29130, CCU-29165, CCU-29166
