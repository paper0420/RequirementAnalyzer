#SYQT-20-0001-0001#

[Charging],[Component_Protection],[AC_Overcurrent]
Test Objective: Confirmation of overcurrent detection of HVAC input current (1 phase)

Requirements

  • 18393 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The "Komponentenschutz und Derating" function shall detect the following protection conditions (detect errors and monitor limit values) in order to adjust operation of the charging electronics without damaging the hardware: - AC undervoltage - DC-HV undervoltage - AC overvoltage - DC-HV overvoltage - Maximum AC current - Maximum DC-HV current - Hardware fault (e.g. short circuit) - Excess temperature - Broken N-Line
  • 18417 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The function "Komponentenschutz und Derating" shall detect an AC overcurrent error if the charger measures an AC current over the maximum AC operating current for a parameterizable period.
  • 18418 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    If an AC overcurrent is detected, the function "Komponentenschutz und Derating" shall set the "Abschaltung Maximalstrom AC" bit in the error status word.
  • 18419 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    These diagnostics shall be performed only if the "Charge" target operating mode (2) is commanded by the "LDK" function on the respective charging module.
  • 18444 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    The "SLE" function shall always observe the phase current limit.
