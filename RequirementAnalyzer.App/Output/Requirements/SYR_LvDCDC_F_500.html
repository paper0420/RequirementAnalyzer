  • 6.1.66 SYR_LvDCDC_F_500: HV overvoltage DTC and buck mode off
  • 6.1.66.1 Description
  • 6.1.66.1.0-1 Brief description:
  • 6.1.66.1.0-2 The LvDCDC stops the buck operation mode when the HV voltage is higher than the p_DCDC_DERAT_MAX_U_HV
  • 6.1.66.1.0-3 Preconditions:
  • 6.1.66.1.0-4 LvDCDC current operating state=Buck, RUNLEVEL >= 30.
  • 6.1.66.1.0-5 Trigger:
  • 6.1.66.1.0-6 Measured_U_HV over 490V for longer 1s.
  • 6.1.66.1.0-7 Input data:
  • 6.1.66.1.0-8 Target_Operating_State = Buck, Target_U_LV = 14V, Measured_U_LV = 14V Measured_U_HV = 490V for 1 s. p_DCDC_MAX_U_HV = 460V. Min_Permitted_HV_Voltage = p_DCDC_DERAT_MAX_U_HV = 490V, Max_Permitted_HV_Power = 4000 W, p_DCDC_CONTINUOUS_I_LV = 221 A, p_DCDC_CONTINUOUS_P_LV = 3100 W, Coolant temperature = 35Â°C
  • 6.1.66.1.0-9 Description of behaviour:
  • 6.1.66.1.0-10 The LvDCDC shall stop operating when a HV voltage is measured which is higher than p_DCDC_DERAT_MAX_U_HV or equal p_DCDC_DERAT_MAX_U_HV for longer than 1s. The LvDCDC shall goes into the error mode. A DTC shall be stored. Per default the value p_DCDC_DERAT_MAX_U_HV is 490V for 1s.
  • 6.1.66.1.0-11 Timing Requirements:
  • 6.1.66.1.0-12 Panasonic Debounce time: 10ms Panasonic Debounce count: 10
