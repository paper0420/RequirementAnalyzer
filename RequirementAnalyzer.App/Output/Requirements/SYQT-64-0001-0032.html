#SYQT-64-0001-0032#

[Thermal_Management],[AKKS],[Error_Handling]
Test Objective: Stop Test Do not detect blockade and terminate abnormally.

Requirements

  • 25228 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The diagnostic function for monitoring the valid range shall detect an exceedance of the valid applicable signal range (upper stop not detected, can only be tested during stop test OPEN)
  • 25253 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the given value is reached without blockade within the traversing range the stop test OPEN has failed.
  • 25254 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the stop test OPEN failed, a debouncing shall start (applicable number of attempts).
  • 25255 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If the blockade cannot be debounced an error message (DFC) shall be generated. (see Table 1)
  • 25256 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    If stop test OPEN was successfull the actuator shall move according to the normal position request.
