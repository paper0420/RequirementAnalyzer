SYR_CIN_330


  • 8.1.37 SYR_CIN_330: Add coding parameter for V-Max CCU
  • 8.1.37.1 Description
  • 8.1.37.1.0-1 Brief description: Coding Parameter for V-MAX needs to be updated in BAC Coding Module. Coding parameter for V-MAX is needed to setup the maximum speed reachable by the car. This system requirement is valid for I20/G26.
  • 8.1.37.1.0-2 Preconditions: CCU in flash mode
  • 8.1.37.1.0-3 Trigger: coding CCU
  • 8.1.37.1.0-4 Input data: CAF file, FA file
  • 8.1.37.1.0-5 Description of behaviour: The V-MAX speed is depending on the SAs, country codes and carlines. For the Coding parameter to be introduced in coding table, the client/server interface of the BAC coding module shall be converted into a S/R interface towards BMW SWC. Acceptance : The coding parameter shall be accessible to BMW SWC
  • 8.1.37.1.0-6 Timing Requirements: NA
  • 8.1.37.1.0-7 Output data: BMW coding parameter signal is set as per FA file
  • 8.1.37.1.0-8 Postconditions: The coding parameter shall be accessible to BMW SWC
  • 8.1.37.1.0-9 Descriptions of exceptions: NA
  • 8.1.37.1.0-10 Dependencies and interactions: NA
