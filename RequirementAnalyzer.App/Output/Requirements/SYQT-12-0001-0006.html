#SYQT-12-0001-0006#

[Charging],[14V_DCDC],[DCDC_Control]
Test Objective: LVDC output current check (Within LVDC output voltage normal range)

Requirements

  • 9924 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The DC/DC converter HV/LV must fullfill the following requirements for the complete unlimited operation range, see section "Operating range of CCU". Differences in the boundary conditions are mentioned explicitely in the requirement text if necessary.
  • 6513 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The CCU must be designed for a minimum LV voltage of 10.5 V for unlimited operation.
  • 6514 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The CCU must be designed for a maximum LV voltage of 15.5 V for unlimited operation.
  • 9988 -
    [Not Changed][partly accepted][SYQT (ENG.10)][Requirement]
    The CCU must be designed for a maximum DC voltage of 460 V in the unlimited operation mode.
  • 14506 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Zustandsmanagement" serves for managing the operating states of the "DC/DC" function: "Init", "Standby", "Buck", "Error" and "Crash". The respective active states are communicated to other functions. The respective operating mode adopted by the subfunction "Zustandsmanagement" corresponds with the operating mode of the "DC/DC" function.
  • 14508 -
    [Not Changed][accepted][SYQT (ENG.10)][Information]
  • 14524 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    The subfunction "Gleichspannungswandlung" shall ensure the transfer of electrical power from the high-voltage to the low-voltage network if the "DC/DC" converter function is in the "Buck" operating mode.
  • 14529 -
    [Not Changed][accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    The subfunction "Gleichspannungswandlung" shall be fully operational in the range between "p_DCDC_MIN_U_LV" and "p_DCDC_MAX_U_LV", in the range between "p_DCDC_MIN_U_HV" and "p_DCDC_MAX_U_HV" and for ambient temperatures in the range between "p_DCDC_MIN_T" and "p_DCDC_MAX_T".
  • 14531 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Gleichspannungswandlung" shall limit the LV current to "p_DCDC_CONTINUOUS_I_LV" if the "DC/DC" converter function is in "Buck" operating mode.
  • 11464 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) connect the development object with resistive load. b) activate and operate the development object at the constant-voltage output mode with the output voltage being a set value Uz0 in the range of output voltage of the development object.
  • 11465 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) adjust the input voltage of the development object such that it varies within ± 15% of the rated value. b) adjust the output current such that it varies within the range between no load to rated current. b) measure actual output voltage Uz.
  • 17648 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) output voltage tolerance must be not more than ± 1% calculated per formula: Of which, ξU -- Output voltage tolerance; Uz -- Actual output voltage (V); Uz0 -- Set output voltage (V).
