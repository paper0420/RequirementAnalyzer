SYR_HMG89


  • 3.1.25 SYR_HMG89:Control of the speed of the electric Waterpumps (EWP23,EWP22)
  • 3.1.25.1 Description
  • 3.1.25.1.0-1 Brief description: CCU has to control the Waterpumps EWP23 and EWP22
  • 3.1.25.1.0-2 Preconditions: Kl30 ON, Ignition ON
  • 3.1.25.1.0-3 Trigger: LIN Wake-Up
  • 3.1.25.1.0-4 Input data: from the BMW interface ---> BMW_pct_CtlEwp1 for EWP23, BMW_pct_CtlEwp2 for EWP22
  • 3.1.25.1.0-5 Description of behaviour: 1- The CCU has to control the speed of the Waterpump EWP23 (P1/130W) via the LIN signal CTR_EWP23_LIN .TARVL_RPM_EWP23_LIN ---> value range [0..254] 255 indicates an invalid value 2- The CCU has to control the speed of theWaterpump EWP22 (P2/80W) via the LIN signal CTR_EWP22_LIN .TARVL_RPM_EWP22_LIN ---> value range [0..254] 255 indicates an invalid value 3- When CCU goes to sleep the pump needs to be deactivated by sending all LIN-Slaves to sleep (Std-LIN-behaviour).

  • 3.1.25.1.0-10 Dependencies and interactions: EPIC# ----> CCU-1466 (EWP23) EPIC# ----> CCU-1465 (EWP22)
