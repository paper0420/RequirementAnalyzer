Test Objective: Confirm that accelerator pedal 2 redundant output falls within the specified accuracy range.

Requirements

  • 24578 -
    [Not Changed][partly accepted][SYIT (ENG.9)][Requirement]
    The voltage value of the redundant acceleration pedal sensor 2 shall be read as "FahrPedal_Signal_U2_Red" with the following accuracy: <1V: +/- 10mV >1V: +/- 1% Safe state: Corresponding Qualifier is set to invalid, FahrPedal_Signal_U2_qi equals invalid FahrPedal_Signal_U2: see reference architecture (last correct value) FTT: 40ms Note: requirements for signal quality are defined in the chapter "Signal Quality". Note: The Signal "FahrPedal_Signal_U2_Red" is relevant only for the Supplier SW. It must be used to check the Signal "FahrPedal_Signal_U2"
