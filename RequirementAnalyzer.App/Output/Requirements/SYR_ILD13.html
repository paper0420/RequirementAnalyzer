  • 1.1.2.4 SYR_ILD13: Compatibility check
  • 1.1.2.4.1 Description
  • 1.1.2.4.1.0-1 Brief description: CCU has to check LBN before transfered to ILD. In case of mismatch, CCU will reject further SW-Update of IKD
  • 1.1.2.4.1.0-2 Preconditions: SW update of ILD is ongoing
  • 1.1.2.4.1.0-3 Trigger: Erase memory -Parameter
  • 1.1.2.4.1.0-4 Input data: LBN parameter of "EraseMemory" command
  • 1.1.2.4.1.0-5 Description of behaviour: CCU will check LBN for correct range, in case of mismatch, negative response will be send to ESYS and SW-Update of ILD is canceled by CCU
