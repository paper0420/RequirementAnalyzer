  • 3.1.26 SYR_HMG90: Cyclical message sequence on LIN
  • 3.1.26.1 Description
  • 3.1.26.1.0-1 Brief description: the CCU has to cyclically send with a period of 800ms a pre-defined sequence of LIN Messages ; this sequence is specified in the official BMW LIN message catalogue and describes the behaviour at run-time of the whole LIN Network in which the CCU is considered as LIN-Master
  • 3.1.26.1.0-2 Preconditions: no physical failure present on H_LIN_4; all the devices connected to the H_LIN_4 Network are working properly
  • 3.1.26.1.0-3 Trigger: LIN Wake-Up
