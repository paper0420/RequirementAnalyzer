#SYQT-25-0001-0004#

[Charging],[Component_Protection],[DC_Undervoltage]
Test Objective: Low Voltage Detection Confirmation of HVDC Output Voltage HV battery -> LV Battery

Requirements

  • 18148 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    The subfunction "Gleichspannungswandlung" shall regulate the HV power consumption so that the high-voltage value does not fall below the value specified by the signal "Spannung_DCDC-Wandler_Hochvoltteil_minimal_Grenze" if the "DC/DC" converter function is in "Buck" operating mode.
  • 14526 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The subfunction "Gleichspannungswandlung" shall prevent the output transfer between high-voltage network and low-voltage network if the "DC/DC" converter function is in the "Standby" operating mode.
  • 18405 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The function "Komponentenschutz und Derating" shall detect a DC-HV undervoltage error if the charger measures a DC voltage below the DC voltage operating limit for a parameterizable period.
  • 18406 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If a DC undervoltage is detected, the function "Komponentenschutz und Derating" shall the "Abschaltung Unterspannung DC-HV" bit error status word.
