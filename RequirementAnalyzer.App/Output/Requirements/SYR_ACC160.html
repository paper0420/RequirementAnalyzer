  • 2.1.22 SYR_ACC160: Component protection - N-L Swap diagnostic Fail Condition [under construction]
  • 2.1.22.1 Description
  • 2.1.22.1.0-1 Brief description:
  • 2.1.22.1.0-2 One of the main root cause of the exploded PFC capacitor issue is a N-L Swap on the EVSE/wallbox diagnosis. In case of CCU failure due to PFC exploded capacitor the setting of a primary DTC is useful to warn the final customer to have the EVSE/Wallbox checked by a qualified electrician.
  • 2.1.22.1.0-3 Preconditions:
  • 2.1.22.1.0-4 • The charging plug is connected AND • the S2 switched is closed AND • Delaytime expired if S2 switch open and closed in short time (before PFC discharge complete) AND • The switch matrix is in 3-phase mode AND • Bit 2 of g_calib_sle_DebugFlag_NLineDisConn = 0 (Enable) AND • The PFC voltage is present AND • The AC voltage is present AND • N Line disconnection error flag is not set for any Line AND • No module is commanded to charge state AND • PFC_OVP error flag set
  • 2.1.22.1.0-5 Trigger:
  • 2.1.22.1.0-6 • PFC Voltage unbalance - L1_PFC_voltage or L3_PFC_voltage > (L2_PFC_voltage + offset) (Condition 2) - L1_PFC_voltage or L2_PFC_voltage > (L3_PFC_voltage + offset) (Condition 3) - L2_PFC_voltage or L3_PFC_voltage > (L1_PFC_voltage + offset) (Condition 1) OR • AC voltage unbalance - L1_AC_voltage or L3_AC_voltage > (L2_AC_voltage + offset) (Condition 2) - L1_AC_voltage or L2_AC_voltage > (L3_AC_voltage + offset) (Condition 3) - L2_AC_voltage or L3_AC_voltage > (L1_AC_voltage + offset) (Condition 1)
  • 2.1.22.1.0-7 Input data:
  • 2.1.22.1.0-8 Signals • PFC Voltage • AC voltage • Target operating state • Charging plug state • S2 state • Switch matrix state • NLine disconnect error state Calibration • Calib_sle_Thresh_Fail_PfcVolt_NLineDisConn (initial 1500 [0.1V]) • Calib_sle_Thresh_Fail_ACVolt_NLineDisConn (initial 500 [0.1V]) • Calib_sle_Offset_PfcVolt_NLineDisConn (initial 800 [0.1 V]) • Calib_sle_Offset_AcVolt_NLineDisConn (initial 500 [0.1 V]) • Calib_sle_DebTime_Fail_NLSwap (initial 50[ms]) • Calib_sle_DebugFlag_NLineDisConn • Calib_S2reqTime (initial 10[s])
