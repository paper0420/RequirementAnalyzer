  • 7.6.1.5 SYR_DBL40: supervise environment - obsolete
  • 7.6.1.5.1 Description
  • 7.6.1.5.1.0-1 Brief description:
  • 7.6.1.5.1.0-2 To see the complete context of whole (car) system, data from outside CCU need to be logged. Sice the raw-data is not sufficient, these data needs to be judged and the judgement has to be provided to external analysis tool. This UC will gather all external supplied data (e.g. via CAN / PLC / analogue measure) and judge it in context of CCU. This “processed” data will be send to external analysis tool on demand. In case to unique environment behaviour/status this UC will generate DLT logging notifications.
