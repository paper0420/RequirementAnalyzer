#SYQT-57-0001-0006#

[Device_Control],[S5_Sensor],[S5_Sensor]
Test Objective: Register DTC code when temperature measured value suddenly changes

Requirements

  • 25096 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    All lines between the temperature sensor coolant radiator outlet and the control unit shall be diagnosed with regard to potentials that do not match the target value switching state.
  • 25100 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The following requirements apply to all line diagnoses:
  • 25101 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - The respective error bit shall be initialized with "FALSE."
  • 25105 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - After debouncing, the relevant error bit shall be set to "True."
