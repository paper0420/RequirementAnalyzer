SYR_CCS_612


  • 1.1.31.26 SYR_CCS_612 : Provide support for selection of Schema Priority via BMW Appl.
  • 1.1.31.26.1 Description
  • 1.1.31.26.1.0-1 Brief description: This SYR describes that the schema priority (DIN or ISO) can be selected by BMW Appl. via SW interface
  • 1.1.31.26.1.0-2 Preconditions: CCU is woken-up. For V2G sequence, PLC bootup and SLAC procedure. Function Control is set to Charging Mode.
  • 1.1.31.26.1.0-3 Trigger: SW interface "BMW_st_CcsIsoPrio BMW_st_CcsStdnPrio" is set
  • 1.1.31.26.1.0-4 Input data: BMW Appl. arxml update in sprint 2211 for the interface.
  • 1.1.31.26.1.0-5 Description of behaviour: This SYR enables that CCU (BSW) shall be able to send the schema priority (DIN or ISO) in SupportedAppProtocol request to EVSE during V2G HLC as selected via BMW-Appl SW. For Schema Priority selection, a new SW interface BMW_st_CcsIsoPrio BMW_st_CcsStdnPrio is defined and shall be set with value as mentioned below. 0: None 1: DIN Priority 1 , no ISO15118 2: ISO15118 Priority 1 , no DIN 3: DIN Priority 1, ISO15118 Priority 2 4: ISO15118 Priority 1 , DIN Priority 2 "0: None" means Schema Priority shall be set as defined in BSW configuration which means DIN as priority 1 and ISO as priority 2 (as per current configuration, however it can be changed if configuration changes). Note: Decision on Schema Priority via BMW Appl. (CIF) component should be available as soon as CCU is wakeup and CIF Appl-SW execution evaluates the priority. Once the priority is decided by CIF, it is not expected to be changed during ECU runtime. However, in BSW, stack will read the available interface value only while preparing SupportedAppProtocol request in V2G HLC.
  • 1.1.31.26.1.0-6 Timing Requirements: NA
  • 1.1.31.26.1.0-7 Output data: SupportedAppProtocol request sent from EV to EVSE during high-level communicaion shall reflect the selected schema priority as set by interface.
  • 1.1.31.26.1.0-8 Postconditions: NA
  • 1.1.31.26.1.0-9 Descriptions of exceptions: NA
  • 1.1.31.26.1.0-10 Dependencies and interactions: SCC stack, SCC adapter, RTE, BMW Appl.
