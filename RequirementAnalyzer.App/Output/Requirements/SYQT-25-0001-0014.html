#SYQT-25-0001-0014#

[Charging],[Component_Protection],[DC_Undervoltage]
Test Objective: Confirm low voltage return of HVDC output voltage (China requirement)

Requirements

  • 6450 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The objekt of development has to restore automatictly the maximum intended output when reentering the high voltage operating condition B1 or B2.
  • 11409 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) activate the development object at input voltage (± 15% of the rated input voltage) and input frequency (50 Hz ± 2%). b) operate the development object at the constant-current state.
  • 17631 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) reduce the load resistance in order that the output voltage of development object is lower than the DC under-voltage protection value as specified by BMW. Note: Resistance depends on the output current and DC undervoltage.
  • 11410 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) the development object must shut down the output and send a bus message when the output voltage is less than the undervoltage protection value. b) the development object must be able to restore output voltage and output current automatically when the malfunction is eliminated.
