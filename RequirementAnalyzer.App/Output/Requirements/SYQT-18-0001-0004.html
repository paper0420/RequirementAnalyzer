#SYQT-18-0001-0004#

[Charging],[Component_Protection],[AC_Overvoltage]
Test Objective: AC Overvoltage Detection Confirmation (China requirement)

Requirements

  • 11398 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) activate and operate the development object at input voltage (± 15% of the rated input voltage) and input frequency (50 Hz ± 2%)
  • 17627 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) adjust the input voltage of the development object such that it exceeds the protection value of input overvoltage as specified by BMW.
  • 11399 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) the development object must shut down the output and send a bus message when the input voltage is more than the overvoltage protection value. b) the development object must be able to restore output voltage and output current automatically when the malfunction is eliminated.
