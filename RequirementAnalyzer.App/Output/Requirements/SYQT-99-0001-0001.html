#SYQT-99-0001-0001#

[Functional_Safety],[Functional_Safety],[Functional_Safety]
Test Objective: HVDC output voltage is limited by limit value Maximum allowed voltage 490V

Requirements

  • 23644 -
    [Not Changed][accepted][SYQT (ENG.10)SYIT (ENG.9)][Requirement]
    The CCU shall limit the output voltage on HVDC (E_U_LT4) to the maximum allowed voltage of 490V. FTT: 1000ms Safe state: Disable the AC charging and inform BMW SW
  • 18395 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The "Komponentenschutz und Derating" function shall stop operation of the charging electronics when a protection condition is present.
  • 18397 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The "Komponentenschutz und Derating" function shall trigger an operating mode change to the "Error" state when a protection condition is present.
  • 18399 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    If a self-protection emergency deactivation takes place, an entry shall be made in the error memory clearly identifying the cause of the error.
  • 11308 -
    [Not Changed][accepted][SYQT (ENG.10)SWQT (ENG.8)][Requirement]
    The "Ansteuerung_Ladegerät_AC/DC_Hochvolt" function shall output the cause of the "Error in the charging electronics" state via the "Grund_Betriebsart-Fehler_Ladegerät-AC/DC_Ist" signal to the LDK function.
  • 17641 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Pre-Condition: a) set the output voltage limitation value of development object as specified by BMW. b) activate the development object.
  • 17642 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Action: a) operate the development object at the constant-current state. b) adjust the load such that the output voltage of the development object increases and gradually reaches the set value for voltage limitation as specified by BMW.
  • 17643 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    Post-Condition: a) during operation, the development object must automatically limit the increment of its output voltage when the output DC voltage attains the voltage limitation setting value as specified by BMW.
