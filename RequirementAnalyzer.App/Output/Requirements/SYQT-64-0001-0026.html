#SYQT-64-0001-0026#

[Thermal_Management],[AKKS],[Function-specific operation]
Test Objective: System check request will be received and system check will be activated.

Requirements

  • 25207 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    A function shall be provided which starts a system check.
  • 25208 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The system check starts a stop test CLOSED and after completion it immediately starts a stop test OPEN.
  • 25211 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    The system check is passed without errors if no early blocking or a too big traveling distance has been detected.
  • 25212 -
    [Not Changed][accepted][SWQT (ENG.8)][Requirement]
    A functionality that allows to perform the system check for each actuator separately OR all actuators with one system check has to be included.
