  • 7.4.2.20 SYR_SUP_190: activate new software / data
  • 7.4.2.20.1 Description
  • 7.4.2.20.1.0-1 Brief description:
  • 7.4.2.20.1.0-2 If flash operation has been finished and verified, this Use-Case will prepare the newly flashed data/software for operation. If software was updated, this Use-Case will just cause “reset” and new software will be booted. If “configuration data” was updated, this Use-Case will inform all affected components to re-read and apply configuration data. It’s NOT desired to reset the CCU after “configuration data” was updated. If the bootloader was updated by “bootloader-updater”, this Use-case will ensure, that the “bootloader-updater” is NOT executed a second time.
  • 7.4.2.20.1.0-3 Preconditions: software was updated successfully
  • 7.4.2.20.1.0-4 Trigger: bootloader stops update an restarts the CCU

  • 7.4.2.20.1.0-6 Description of behaviour: SW will be checked and adapted/configured on demand
