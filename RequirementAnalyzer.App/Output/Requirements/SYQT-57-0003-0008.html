#SYQT-57-0003-0008#

[Device_Control],[S5_Sensor],[S5_Sensor]
Test Objective: Confirmation that error release and DTC registration cancellation will be canceled when the temperature difference between S5 sensor and WJ sensor returns to normal.

Requirements

  • 25087 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The diagnostic function shall monitor the signal of the temperature sensor coolant radiator outlet for plausible values within the physical limits of the out-of-range check.
  • 25089 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The diagnostic function shall generate status information about the current signal quality of the temperature sensor coolant radiator outlet.
  • 25091 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The status information about the signal quality shall be published.
  • 25105 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - After debouncing, the relevant error bit shall be set to "True."
  • 25106 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    If the error is deleted and if the reason for the error is omitted together with fulfillment of the remedy condition, the error bit shall be reset to "FALSE."
  • 25107 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - The remedy condition for each electrical error type shall be implemented applicably on an event- or time-triggered basis.
  • 25114 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    - The default value of the remedy time shall be 3 seconds.
