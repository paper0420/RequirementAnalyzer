  • 1.1.29.2.1.0-5 Description of behaviour: When the DC-China is Enabled. Following action from J1939 BSW package is expected. 1. CCU should claim address 244(F4H) and should establish point to point connection with External Charger (EVSE) with address 86(56H). This activity is expected by J1939NM 2. The Request handling message should be expected to establish data link layer. It should be expected by J1939RM BSW package. 3. Communciation over CAN via J1939 protocol is available to LIM-C via ComAdaptor CAN Specification: - CAN 2.0B Active, Extended 29-bits - Baudrate 250kbps
