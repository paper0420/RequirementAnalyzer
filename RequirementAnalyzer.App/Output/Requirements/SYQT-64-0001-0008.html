#SYQT-64-0001-0008#

[Thermal_Management],[AKKS],[basic action]
Test Objective: Move the flap position with the BMW instruction.

Requirements

  • 25171 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The stepper motor position (target position) shall be calculated.
  • 25176 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The status "adopt stepper motor position" shall be calculated and forwarded to the respective active cooling air flap actuator.
  • 25181 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The stepper motor mode (normal, service, stop) shall be calculated.
  • 25186 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The speed (TURN_V_AKKS_LIN) shall be set in a configurable manner depending on the requirement (stop test OPEN, stop test CLOSED, opening, closing).
  • 25187 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The torque delimiter (SMOT_TORQ_AKKS_LIN) shall be set in a configurable manner depending on the requirement (stop test OPEN, stop test CLOSED, opening, closing).
  • 25188 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    It shall be possible to specify the AKKS "Target programming" by means of an application.
  • 25190 -
    [Not Changed][accepted][SYQT (ENG.10)][Requirement]
    The full AKKS function must be guaranteed in all PAD-states (see _17_783). Via application the function shall be deactivateable in certain PAD-states.
