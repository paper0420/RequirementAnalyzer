  • 3.1.30 SYR_HMG94: DTC 0x224103 "Cooling-pump HVB - Switch-off due to Overcurrent"
  • 3.1.30.1 Description
  • 3.1.30.1.0-1 Brief description: the CCU has to be able to detect the failure "Switch-off due to Overcurrent" of the Waterpump EWP22 (Cooling-pump of the High-Voltage Battery)
  • 3.1.30.1.0-2 Preconditions: Kl15 ON and LIN Communication between the Waterpump EWP22 and the LIN-Master CCU is correctly working
  • 3.1.30.1.0-3 Trigger: ST_EWP22_LIN.ERR_ST_OCU_EWP22_LIN=0x01;
  • 3.1.30.1.0-4 Input data: LIN Signal ERR_ST_OCU_EWP22_LIN
  • 3.1.30.1.0-5 Description of behaviour: The Waterpump EWP22 will signalize to the CCU via LIN the presence of the failure "Switch-off due to Overcurrent". The failure will be detected when the signal ERR_ST_OCU_EWP22_LIN=0x01. As soon as this error condition is detected (after the expiration of a debouncing time whose value has still to be provided by BMW) an entry with Code 0x224103 will be written in the Error Memory
  • 3.1.30.1.0-6 Timing Requirements: Debouncing Time----> 5000 ms (starting from the 13th message) Healing Time ----> 2000 ms
  • 3.1.30.1.0-7 Output data: the DTC with code 0x224103

  • 3.1.30.1.0-10 Dependencies and interactions: dependency to: a. the LIN signal ERR_ST_OCU_EWP22_LIN b. LIN Communication (in case of physical failures on LIN this failure cannot be detected)
