SYR_COA_072


  • 7.9.1.15 SYR_COA_072: Set DTC when error is present after debouncing
  • 7.9.1.15.1 Description
  • 7.9.1.15.1.0-1 Brief description: This UC gives brief description of the COM Adapter setting DTC when a error is present after debouncing time then the Qualifier is set to Error and COM Adapter records DTC through DEM events.
  • 7.9.1.15.1.0-2 Preconditions: RUNLEVEL>=30
  • 7.9.1.15.1.0-3 Trigger: When Debouncing Qualifier status is detected as "ERROR".
  • 7.9.1.15.1.0-4 Input data: Input signal with error present
  • 7.9.1.15.1.0-5 Description of behaviour: When error is detected by debouncing for specified duration then the COM Adapter shall store the corresponding DTC through DEM "SetEventStatus" immediately.. ComAdapter shall read the debouncing counters for CRC and Alive from Message catalogue and run the debouncing mechanism.
  • 7.9.1.15.1.0-6 Timing Requirements: Defect condition: DEF debounce counter * cycle time of the scheduling table in ms
  • 7.9.1.15.1.0-7 Output data: DTC code recorded in memory when qualifier ststus is "Error"
  • 7.9.1.15.1.0-8 Postconditions: None
  • 7.9.1.15.1.0-9 Descriptions of exceptions: None
  • 7.9.1.15.1.0-10 Dependencies and interactions: Interaction with LIM C and BMW components
